# Évaluation du gain explicatif — cycle métrologique SI

## Statut

Ce document clôt le transfert du cycle SI.

Documents associés :

- [architecture-relationnelle-metrologique-si.md](architecture-relationnelle-metrologique-si.md)
- [verification-metrologique-si.md](verification-metrologique-si.md)

## 1. Question d’évaluation

> La description relationnelle du SI apporte-t-elle davantage que le rappel selon lequel certaines constantes ont une valeur exacte par définition ?

## 2. Résultat principal

```text
Cycle SI concluant comme cas contrasté.
```

Le gain ne réside pas dans la découverte que les valeurs sont exactes. Il réside dans la séparation de quatre niveaux :

```text
grandeur ou référence physique
valeur numérique dans un système d’unités
définition institutionnelle
réalisation expérimentale.
```

## 3. Architecture définitionnelle

Le SI constitue une architecture parce qu’il coordonne :

- sept valeurs numériques exactes ;
- les dimensions et unités associées ;
- les règles de dérivation de toutes les unités ;
- des mises en pratique ;
- des comparaisons internationales ;
- l’expression normalisée des incertitudes.

Cette architecture est opératoire et institutionnelle. Elle ne constitue pas une théorie physique expliquant l’origine des constantes.

## 4. Hétérogénéité comme résultat positif

Le cycle montre que :

```text
constante définissante du SI
```

est une fonction commune tenue par des objets physiquement hétérogènes.

- `c` possède un rôle relativiste ;
- `h` un rôle quantique ;
- `e` un rôle électromagnétique ;
- `k_B` un rôle thermodynamique ;
- `N_A` un rôle de comptage et de quantité de matière ;
- `Delta nu_Cs` une fonction de référence atomique ;
- `K_cd` une fonction photométrique conventionnelle.

Le terme « définissante » décrit leur rôle dans le SI, non leur essence physique.

## 5. Pouvoir explicatif

La reprise explique :

- pourquoi une valeur numérique peut être exacte alors qu’une réalisation reste incertaine ;
- pourquoi la fixation de `h` ne constitue pas une mesure parfaite de `h` ;
- pourquoi les incertitudes se reportent sur des constantes dérivées ;
- pourquoi des compositions deviennent exactes numériquement ;
- pourquoi les sept constantes n’ont pas besoin d’avoir la même nature physique pour former un système cohérent ;
- pourquoi une révision métrologique peut changer des définitions sans modifier les phénomènes.

## 6. Contrefactuels

Le cycle produit plusieurs contrefactuels :

- changer d’unités modifie les nombres dimensionnés sans modifier les rapports physiques invariants ;
- modifier une définition future de la seconde modifie le système de référence, non la dynamique temporelle ;
- une mauvaise réalisation augmente l’incertitude sans changer la valeur définissante ;
- une variation physique d’un rapport sans dimension ne peut être annulée par convention.

Ces cas donnent au dossier un pouvoir explicatif supérieur à une simple liste de valeurs exactes.

## 7. Réponse à la fonction de fixité

Le SI confirme que plusieurs types d’objets peuvent tenir une même fonction de fixité :

> fournir des points fixes numériques permettant la construction et la reproduction d’un système de mesure.

Cette conclusion soutient l’hypothèse selon laquelle « constante » peut parfois désigner une fonction tenue dans une architecture plutôt qu’une espèce naturelle unique.

Mais elle impose une limite :

```text
fonction de fixité commune
≠
nature physique commune.
```

## 8. Comparaison des trois cycles

| Cycle | Type de structure | Fixité dominante | Source du contrôle |
|---|---|---|---|
| Higgs–Yukawa | architecture physique et représentationnelle | échelles, couplages, spectres, orientations | lagrangien, régime, accès |
| EFT basse énergie | architecture méthodologique de validité | stabilité des prédictions sous dépendances compensées | power counting, matching, RG |
| SI | architecture définitionnelle | exactitude numérique et comparabilité | décision normative, dérivation, réalisation |

Le vocabulaire de « stabilisation » ne peut plus être employé sans préciser lequel de ces niveaux est visé.

## 9. Nouveau dégagement conceptuel

Les trois cycles permettent de distinguer :

### Invariance

Une grandeur ou une relation demeure inchangée sous des transformations physiques.

### Validité contrôlée

Une description demeure prédictive dans un domaine avec une erreur estimable.

### Exactitude définitionnelle

Une valeur numérique est fixée sans incertitude dans un système d’unités.

### Robustesse de réalisation

Des procédures indépendantes reproduisent une unité ou un résultat dans les incertitudes déclarées.

Ces quatre propriétés sont logiquement distinctes.

## 10. Test de transférabilité

| Critère | Résultat |
|---|---|
| Chaînes distinguées | oui : histoire empirique, définition, réalisation, rôle physique |
| Relations typées | oui |
| Contrefactuels | oui |
| Expliqué / défini / mesuré séparés | oui |
| Refus de faux qualificatifs | oui : homogénéité physique, mesure parfaite, `K_cd` fondamentale |
| Gain indépendant du vocabulaire interne | oui : structure officielle du SI |

## 11. Corrections définitives au corpus

Les formulations suivantes doivent être abandonnées ou corrigées :

```text
h = opérateur de relation quantique
```

à remplacer par :

```text
h = constante de proportionnalité et échelle d’action.
```

```text
e = quantum universel de toute charge
```

à remplacer par une formulation tenant compte des charges fractionnaires des quarks.

```text
exact dans le SI = empiriquement connu sans incertitude
```

à refuser.

## 12. Verdict

```text
Architecture métrologique définitionnelle : confirmée.
Espèce physique unique des constantes définissantes : refusée.
Gain explicatif : confirmé.
```

## 13. Conséquence pour la question directrice

Le cycle apporte une réponse partielle à :

> Qu’est-ce qu’une constante ?

Dans le contexte métrologique :

> Une constante définissante est une grandeur ou référence sélectionnée dont la valeur numérique exacte sert de point fixe à la construction d’un système d’unités.

Cette réponse est locale. Elle ne définit pas la constante physique en général.

## 14. Décision de transfert

Le passage à la cosmologie est autorisé, mais sous une prudence accrue.

Le cycle cosmologique devra distinguer :

- paramètres de loi ;
- paramètres d’état ;
- fonctions dynamiques ;
- conditions initiales ;
- observables reconstruites ;
- tensions et bornes ;
- dépendance au modèle.

Il devra probablement exclure davantage d’objets qu’il n’en admet.

## 15. Formule de clôture

> Le SI montre qu’une fixité peut être exacte sans être dynamique, universelle sans être ontologiquement homogène, et physiquement appuyée sans être une explication de la nature.
