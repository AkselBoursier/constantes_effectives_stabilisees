# Cycle metrologique SI v0.1 - c, h, e

## Constantes definissantes, role physique et architecture metrologique

### 1. Fonction de la fiche

Cette fiche ouvre l'extension metrologique SI annoncee par la carte consolidee v1.0.

Question directrice :

> Comment classer c, h et e sans reduire leur statut a leur exactitude SI, ni perdre leur role physique propre ?

L'objectif est de tester l'architecture metrologique SI sur trois constantes definissantes centrales :

```text
c
h
e
```

La difficulte est que ces grandeurs ont deux lectures simultanees :

```text
1. elles ont une valeur numerique exacte dans le SI ;
2. elles gardent un role physique structurant hors de la seule convention metrologique.
```

La fiche doit donc eviter deux erreurs symetriques :

```text
reduire c, h, e a de simples conventions ;
traiter leur exactitude SI comme une mesure physique plus precise.
```

### 2. Point de depart SI

Dans le SI actuel, les unites sont definies a partir de sept constantes definissantes.

Parmi elles :

```text
c : vitesse de la lumiere dans le vide
h : constante de Planck
e : charge elementaire
```

Valeurs exactes dans le SI :

| Constante | Valeur exacte SI | Unite |
|---|---|---|
| c | 299 792 458 | m s^-1 |
| h | 6.626 070 15 x 10^-34 | J s |
| e | 1.602 176 634 x 10^-19 | C |

Decision methodologique :

```text
exact dans le SI = exact par definition du systeme d'unites.
```

Cela ne signifie pas :

```text
mesure empirique parfaite ;
explication physique complete ;
absence de role physique propre.
```

### 3. Decision centrale

Le test confirme l'architecture metrologique SI comme architecture inter-familles.

Formule :

```text
Architecture metrologique SI =
Convention
+ Composition exacte sous controle v1.3
+ Echelle
+ Realisation pratique
+ Role physique conserve
```

La decision de rang est :

```text
metrologique SI = architecture inter-familles
```

et non :

```text
famille metrologique superieure
```

Les constantes definissantes appartiennent localement a la famille Convention, mais leur role physique peut mobiliser composition, echelle, couplage ou relation qualifiee selon le cas.

Mise a jour v1.3 :

```text
une relation exacte d'ecriture ne suffit pas a classer une grandeur dans Relation ;
il faut d'abord identifier la forme logique, le regime SI, l'acces et l'architecture.
```

### 4. Corpus traite

| Grandeur | Statut SI | Role physique principal | Risque controle |
|---|---|---|---|
| c | Constante definissante primaire | Relation espace-temps / echelle causale | Reduire c a une conversion d'unites |
| h | Constante definissante primaire | Quantum d'action / relation energie-frequence | Confondre exactitude SI et explication quantique |
| e | Constante definissante primaire | Quantum de charge / entree du couplage electromagnetique | Confondre charge dimensionnee et couplage sans dimension |

Ce tableau montre que les trois constantes ont le meme statut metrologique, mais pas le meme role physique.

### 4bis. Rang 0 et composition exacte

Le rang pre-familial permet de distinguer :

```text
valeur definissante
composition exacte
relation physique structurante
couplage sans dimension
realisation pratique
```

Dans l'architecture SI, `K_J`, `R_K` et `R` sont d'abord des compositions exactes heritees du systeme de definition.

Elles ne doivent pas etre classees dans Relation par defaut.

Decision :

```text
composition exacte = forme logique de stabilisation ;
Relation eventuelle = seulement si la liaison joue une fonction irreductible au-dela de cette composition.
```

### 5. c : constante definissante et structure espace-temps

Statut SI :

```text
c est une constante definissante primaire.
```

Elle fixe la valeur numerique exacte :

```text
c = 299 792 458 m s^-1
```

Lecture metrologique :

```text
la definition du metre est reliee a la seconde par la valeur fixee de c.
```

Lecture physique :

```text
c stabilise une relation structurelle entre temps, longueur et causalite relativiste.
```

Classement local :

| Critere | Classement |
|---|---|
| Famille principale dans l'architecture SI | Convention |
| Fonction physique associee | Relation espace-temps / echelle causale |
| Stabilisation conventionnelle | Definissante, exacte |
| Regime physique | Relativite, propagation relativiste, structure causale |
| Regime d'acces | Realisations metrologiques de longueur et de temps |

Decision :

```text
c n'est pas seulement une convention.
```

Mais dans le SI :

```text
son exactitude numerique vient de la definition du metre, non d'une mesure de c devenue parfaite.
```

Formulation stabilisee :

> c est une constante definissante dans l'architecture SI et une relation structurante dans l'architecture relativiste.

### 6. h : constante definissante et quantum d'action

Statut SI :

```text
h est une constante definissante primaire.
```

Elle fixe la valeur numerique exacte :

```text
h = 6.626 070 15 x 10^-34 J s
```

Lecture metrologique :

```text
la valeur fixee de h contribue a definir le kilogramme par l'unite d'action J s = kg m^2 s^-1.
```

Lecture physique :

```text
h stabilise le quantum d'action et les relations energie-frequence et impulsion-longueur d'onde.
```

Relations typiques :

```text
E = h nu
p = h / lambda
hbar = h / (2 pi)
```

Classement local :

| Critere | Classement |
|---|---|
| Famille principale dans l'architecture SI | Convention |
| Fonction physique associee | Echelle d'action + relation quantique |
| Stabilisation conventionnelle | Definissante, exacte |
| Regime physique | Physique quantique, spectroscopie, metrologie quantique |
| Regime d'acces | Realisations du kilogramme, balances de Kibble, relations quantiques |

Decision :

```text
h ne doit pas etre classe seulement comme Convention hors contexte.
```

Dans l'architecture SI, h est Convention. Dans le regime quantique, h fonctionne comme echelle d'action et comme relation entre grandeurs.

Formulation stabilisee :

> h est exacte comme constante definissante du SI, mais son role physique reste celui d'une echelle d'action et d'un operateur de relation quantique.

### 7. e : constante definissante et quantum de charge

Statut SI :

```text
e est une constante definissante primaire.
```

Elle fixe la valeur numerique exacte :

```text
e = 1.602 176 634 x 10^-19 C
```

Lecture metrologique :

```text
la valeur fixee de e contribue a definir l'ampere et le coulomb.
```

Lecture physique :

```text
e stabilise le quantum de charge electrique.
```

Mais :

```text
e dimensionne n'est pas, seul, le couplage electromagnetique sans dimension.
```

Le couplage electromagnetique lisible comme grandeur sans dimension est plutot :

```text
alpha
```

avec une relation de type :

```text
alpha = e^2 / (4 pi epsilon_0 hbar c)
```

Dans le SI revise, c, h et e sont exacts, mais alpha reste determinee experimentalement ; l'incertitude se reporte notamment sur epsilon_0 et mu_0.

Classement local :

| Critere | Classement |
|---|---|
| Famille principale dans l'architecture SI | Convention |
| Fonction physique associee | Echelle de charge / entree du couplage electromagnetique |
| Stabilisation conventionnelle | Definissante, exacte |
| Regime physique | Electromagnetisme, charge elementaire, metrologie electrique |
| Regime d'acces | Definition de l'ampere, comptage de charge, effets quantiques |

Decision :

```text
e ne doit pas etre classe directement comme Couplage sans qualification.
```

La bonne lecture est :

```text
e : constante definissante et quantum de charge.
alpha : couplage electromagnetique sans dimension.
```

Formulation stabilisee :

> e est exacte comme constante definissante du SI ; son role physique est de fixer le quantum de charge, tandis que la force de l'interaction electromagnetique reste mieux lue par alpha.

### 8. Relations exactes par composition

Le couple h / e stabilise aussi des constantes de metrologie electrique.

Exemples :

```text
K_J = 2 e / h
R_K = h / e^2
```

Lecture :

```text
K_J : constante de Josephson
R_K : constante de von Klitzing
```

Comme h et e sont exacts dans le SI, ces relations donnent des constantes exactes par composition dans ce systeme.

Classement :

```text
K_J : composition exacte dans l'architecture SI
R_K : composition exacte dans l'architecture SI
```

Decision :

```text
l'architecture SI ne contient pas seulement des constantes definissantes primaires ; elle contient aussi des constantes exactes par composition.
```

Cela prolonge le cas deja stabilise :

```text
R = N_A k_B
```

### 9. Difference entre exactitude SI et role physique

La fiche confirme une regle centrale :

```text
Exactitude conventionnelle n'est pas precision empirique.
```

Dans le SI :

```text
c, h et e ont des valeurs numeriques exactes.
```

Mais :

```text
les realisations pratiques gardent des incertitudes ;
les grandeurs derivees peuvent rester experimentales ;
les theories physiques ne sont pas prouvees par l'exactitude SI ;
les roles physiques ne sont pas epuises par la convention.
```

Exemples :

```text
c exact dans le SI ne supprime pas les questions de propagation dans les milieux.
h exact dans le SI ne donne pas une interpretation unique de la quantification.
e exact dans le SI ne fixe pas alpha.
```

Regle :

```text
Toujours distinguer valeur definissante, realisation pratique et contenu physique.
```

### 10. Architecture metrologique SI et familles

Les trois constantes ont la meme famille principale dans l'architecture SI :

```text
Convention
```

Mais leurs roles physiques secondaires divergent.

| Grandeur | Famille SI | Fonction physique associee | Familles ou fonctions mobilisees |
|---|---|---|---|
| c | Convention | Relation espace-temps / echelle causale | Relation qualifiee, Echelle |
| h | Convention | Quantum d'action / relation energie-frequence | Echelle, relation quantique |
| e | Convention | Quantum de charge / entree de alpha | Echelle, Couplage sous reformulation |

Decision :

```text
Convention peut etre la famille principale dans l'architecture SI sans effacer les fonctions physiques associees.
```

Cette decision renforce la methode generale :

```text
famille principale = fonction locale dominante dans le contexte choisi
architecture = articulation de fonctions dans un regime
rang 0 = forme logique de stabilisation avant la famille
```

### 11. Regime physique et regime d'acces

| Grandeur | Regime physique | Regime d'acces |
|---|---|---|
| c | Relativite, propagation dans le vide, structure causale | Realisations de longueur et de temps, definitions SI |
| h | Physique quantique, action, spectroscopie | Realisations du kilogramme, balances de Kibble, frequences |
| e | Charge elementaire, electromagnetisme, metrologie electrique | Definition de l'ampere, comptage de charges, effets Josephson et Hall quantique |

Regle :

```text
Le regime d'acces SI ne remplace pas le regime physique.
```

Il le rend operatoire dans un systeme de mesure.

### 13. Ce que la fiche confirme

La fiche confirme six decisions.

```text
1. L'architecture metrologique SI est bien une architecture inter-familles.
2. Convention reste la famille principale pour les constantes definissantes.
3. Une constante definissante peut garder un role physique fort.
4. Exactitude SI n'est pas precision empirique.
5. Les constantes exactes par composition prolongent les constantes definissantes primaires.
6. e ne doit pas etre confondu avec le couplage electromagnetique sans dimension alpha.
7. La composition exacte releve d'abord du rang 0, non de Relation par defaut.
```

### 14. Ce que la fiche ne confirme pas

La fiche ne confirme pas :

```text
une famille Metrologie ;
une famille Definition ;
une famille Conversion ;
une reduction de c, h et e a de simples conventions ;
une identification de e au couplage electromagnetique complet ;
une exactitude empirique absolue des realisations pratiques.
```

Elle ne donne pas non plus :

```text
une philosophie complete du SI ;
une theorie de la quantification ;
une theorie du couplage electromagnetique ;
une analyse complete des constantes electriques derivees.
```

### 15. Formule de travail

Formule longue :

> Dans l'architecture metrologique SI, c, h et e sont des constantes definissantes exactes par convention, mais elles conservent des roles physiques distincts : relation espace-temps, quantum d'action et quantum de charge.

Formule courte :

> Une constante definissante peut etre exacte dans le SI sans etre seulement conventionnelle en physique.

### 19. Formule de cloture

Le test est reussi s'il permet d'ecrire :

```text
c est exact dans le SI, mais n'est pas seulement une convention.
h est exact dans le SI, mais reste quantum d'action.
e est exact dans le SI, mais alpha reste le couplage sans dimension.
K_J et R_K sont exacts par composition.
L'architecture SI n'est pas une famille.
La composition exacte n'est pas Relation par defaut.
```

Formule finale :

> L'architecture metrologique SI stabilise des constantes en fixant leurs valeurs numeriques dans un systeme d'unites, mais elle ne supprime ni leur regime physique, ni leurs relations, ni leurs limites d'interpretation.
