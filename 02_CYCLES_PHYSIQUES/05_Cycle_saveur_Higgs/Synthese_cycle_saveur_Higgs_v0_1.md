# Synthese du cycle Saveur-Higgs v0.1

## Architecture constitutive, echelle de brisure et orientations de saveur

### 1. Fonction de cette synthese

Cette synthese met le cycle Saveur-Higgs au meme rang documentaire que les autres cycles physiques actifs.

Elle ne remplace pas les fiches historiques sur :

```text
v
Yukawa
CKM
Wolfenstein
Delta m^2
angles de melange leptoniques
masse absolue des neutrinos
```

Elle extrait leur consequence methodologique principale :

```text
Saveur-Higgs = architecture constitutive inter-familles
```

et non :

```text
famille Saveur-Higgs
```

### 2. Question directrice

La question du cycle est :

> Qu'est-ce qui devient stabilise dans le secteur Saveur-Higgs lorsque l'echelle de brisure, les matrices de couplage, les masses et les orientations deviennent solidaires ?

La reponse n'est pas une constante unique.

Le cycle stabilise une structure :

```text
echelle de brisure
+ couplages de texture
+ diagonalisation
+ spectres de masses
+ orientations observables
```

### 3. Decision centrale

La decision de rang est :

```text
Architecture Saveur-Higgs = architecture constitutive confirmee.
```

Elle articule plusieurs familles ou roles :

```text
Echelle
Couplage
Orientation
Spectre stabilise
Relation constitutive sous audit
Borne ou projection pour certains neutrinos
```

Elle ne cree pas une famille superieure.

### 4. Trajectoire constitutive

Le cycle n'est pas seulement un tableau de grandeurs juxtaposees.

Il a une trajectoire de stabilisation :

```text
regime electrofaible non brise
-> brisure electrofaible
-> matrices de Yukawa
-> matrices de masse
-> diagonalisation
-> masses physiques
-> orientations CKM / PMNS
-> manifestations transitionnelles ou oscillatoires
```

Cette temporalite est conceptuelle.

Elle ne pretend pas expliquer l'origine profonde des Yukawa.

Elle indique le regime dans lequel les grandeurs deviennent definissables, mesurables ou reconstructibles.

### 5. Rang 0 et refus du classement relationnel automatique

Le rang 0 v1.3 est decisif pour ce cycle.

Plusieurs objets ont une forme relationnelle :

```text
m_f = y_f v / sqrt(2)
M_f = Y_f v / sqrt(2)
V_CKM = U_L^u dagger U_L^d
U_PMNS = U_L^e dagger U_nu
Delta m^2_ij = m_i^2 - m_j^2
```

Mais une forme relationnelle ne suffit pas a classer la grandeur dans la famille Relation.

Decision :

```text
Relation est presente comme forme logique ou liaison constitutive,
mais le classement robuste passe par l'architecture Saveur-Higgs.
```

### 6. Corpus stabilise

| Grandeur | Rang 0 | Famille ou role principal | Fonction dans le cycle |
|---|---|---|---|
| `v` | Valeur d'echelle | Echelle | Echelle commune du regime electrofaible brise |
| `Y_f` | Matrice | Couplage | Couplages de texture distribuant l'echelle |
| `m_f` | Valeur spectrale issue d'une relation | Spectre stabilise | Masse physique apres brisure et diagonalisation |
| `CKM` | Matrice d'orientation | Orientation | Desalignement transitionnel du secteur quark |
| Parametres de Wolfenstein | Coordonnees hierarchiques | Role contextuel | Parametrisation locale de CKM |
| `PMNS` | Matrice d'orientation | Orientation | Desalignement oscillatoire du secteur neutrino massif |
| `Delta m^2` | Ecart spectral | Dephasage oscillatoire | Grandeur accessible par oscillations |
| Masse absolue neutrino | Borne / projection | Ancrage spectral sous audit | Fiche prudente produite, interface Saveur-Higgs / cosmologie |

### 7. Regime physique et regimes d'acces

Le regime physique principal est :

```text
regime electrofaible brise
```

Les regimes d'acces sont heterogenes :

| Grandeur | Regime d'acces | Prudence |
|---|---|---|
| `v` | Inference via `G_F`, masses faibles et ajustements electrofaibles | Forte dans le cadre standard |
| Yukawa charges | Reconstruction via masses, Higgs et schema de renormalisation | Origine ouverte |
| Masses de quarks | Fits et schemas de masse | Robustesse dependante du schema |
| CKM | Desintegrations faibles, mesons, fits globaux | Forte avec tensions locales possibles |
| PMNS | Oscillations neutrino | Forte pour angles principaux, incomplete pour phases |
| `Delta m^2` | Oscillations neutrino | Acces aux ecarts, non a l'ancrage absolu |
| Masse absolue neutrino | Beta decay, cosmologie, double beta sans neutrino selon hypotheses | Bornee, modele-dependante, fiche prudente dediee |

Regle :

```text
une grandeur Saveur-Higgs doit toujours etre lue avec son regime physique et son regime d'acces.
```

### 8. Formes internes du cycle

Le cycle stabilise quatre formes internes.

| Forme | Grandeurs | Fonction |
|---|---|---|
| Echelle commune | `v` | Rend possible la lecture des masses dans le regime brise |
| Distribution par couplages | `Y_f` | Organise les textures et hierarchies fermioniques |
| Stabilisation spectrale | `m_f`, `Delta m^2` | Produit des valeurs ou ecarts accessibles |
| Orientation entre bases | CKM, PMNS, phases | Rend observables les desalignements |

Ces formes ne se remplacent pas.

Elles sont solidaires dans l'architecture.

### 9. Ce que le cycle confirme

Le cycle confirme :

```text
1. Saveur-Higgs est une architecture constitutive.
2. Cette architecture n'est pas une famille.
3. Relation ne doit pas absorber les masses, matrices et ecarts.
4. Orientation reste une famille forte pour CKM et PMNS.
5. Les neutrinos exigent une prudence d'acces et de modele.
6. La temporalite pertinente est une temporalite de constitution.
```

### 10. Ce que le cycle ne confirme pas

Le cycle ne confirme pas :

```text
une origine dynamique des Yukawa ;
une explication des hierarchies fermioniques ;
une resolution de la nature Dirac ou Majorana des neutrinos ;
une valeur absolue stabilisee des masses neutrino ;
une famille Saveur-Higgs ;
une famille Relation pour toute ecriture de masse ou de melange.
```

### 11. Stress test d'architecture

| Test | Resultat |
|---|---|
| Plusieurs familles sont-elles solidaires ? | Oui : Echelle, Couplage, Orientation, roles spectraux |
| Le retrait de `v` modifie-t-il le reseau ? | Oui : plus d'echelle commune de brisure |
| Le retrait des Yukawa modifie-t-il le reseau ? | Oui : plus de distribution entre fermions |
| Le retrait de la diagonalisation modifie-t-il le reseau ? | Oui : plus de masses physiques ni d'orientations lisibles |
| Les regimes d'acces sont-ils explicites ? | Oui, mais heterogenes |
| L'architecture se reduit-elle a Relation ? | Non |

Conclusion :

```text
Architecture Saveur-Higgs confirmee comme architecture constitutive.
```

### 12. Formule issue du cycle

Formule longue :

> Une grandeur Saveur-Higgs n'est pas seulement une valeur ou une relation ; elle prend statut dans une trajectoire constitutive ou l'echelle de brisure, les couplages, les spectres et les orientations deviennent solidaires.

Formule courte :

> Saveur-Higgs stabilise une constitution, non une famille.
