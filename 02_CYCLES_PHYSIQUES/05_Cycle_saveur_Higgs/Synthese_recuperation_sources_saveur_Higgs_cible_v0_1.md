# Synthese de recuperation des sources Saveur-Higgs cible v0.1

## Bloc v, Yukawa, CKM, Wolfenstein

### 1. Fonction

Cette note recupere le bloc DOCX cible utile a l'architecture Saveur-Higgs
avant l'ouverture du cercle 2.

Sources remontees :

```text
Source_docx_Fiche_v_v0_1.md
Source_docx_Fiche_Yukawa_v0_1.md
Source_docx_Fiche_CKM_v0_1.md
Source_docx_Fiche_Wolfenstein_v0_1.md
Source_docx_Synthese_cycle_CKM_v0_1.md
```

Le but n'est pas de rouvrir tout le cycle Saveur-Higgs.

Le but est de verifier les poles actifs :

```text
echelle constitutive,
textures de couplage,
orientation,
parametrisation hierarchique,
et acces phenomenologique.
```

### 2. Recuperation principale

Le bloc confirme une trajectoire constitutive.

```text
champ de Higgs dans le vide
  -> echelle electroweak brisee
  -> couplages de Yukawa
  -> spectres de masses
  -> diagonalisation
  -> matrices d'orientation
  -> observables de transition et de violation CP.
```

Lecture par source :

| Source | Recuperation utile | Traduction v1.3 |
|---|---|---|
| `v` | Valeur du champ de Higgs dans le vide ; echelle du regime brise | Echelle constitutive, non etiquette autonome |
| Yukawa | Couplages et textures libres du Modele standard | Couplages de texture, origine ouverte |
| CKM | Orientation des etats de saveur des quarks | Orientation reconstruite par transitions |
| Wolfenstein | Coordonnees hierarchiques de CKM | Parametrisation locale, pas nouvelle categorie |
| Synthese CKM | Confirmation d'une orientation au-dela des neutrinos | Extension de la preuve Saveur-Higgs |

### 3. Correction de vocabulaire

Les sources utilisent encore des formulations anciennes.

Elles doivent etre traduites positivement dans la methode actuelle.

```text
ancien vocabulaire : regime generateur, famille relationnelle, lien interne
traduction active  : echelle constitutive, couplage de texture, orientation,
                     parametrisation, architecture Saveur-Higgs.
```

La presence de formules ou de dependances internes ne suffit pas a produire
un rang taxonomique autonome.

La lecture robuste est :

```text
les dependances internes du secteur sont absorbees par l'architecture
constitutive Saveur-Higgs, sauf lorsqu'une fonction locale resiste au
criblage.
```

### 4. Points a conserver pour le cercle 2

Le cercle 2 devra conserver quatre points.

```text
1. `v` n'est pas une masse ni un couplage : c'est une echelle constitutive.
2. Les Yukawa fixent des intensites et textures, mais leur origine reste ouverte.
3. CKM stabilise une orientation par reconstruction phenomenologique.
4. Wolfenstein donne des coordonnees utiles, non une structure nouvelle.
```

Le bloc confirme aussi un point methodologique :

```text
une architecture peut etre plus fondamentale localement que le classement
fonctionnel d'une grandeur isolee.
```

### 5. Effet sur l'architecture Saveur-Higgs

L'architecture Saveur-Higgs sort renforcee.

Elle ne repose pas seulement sur des grandeurs juxtaposees.

Elle decrit une chaine :

```text
regime brise -> couplages -> masses -> orientations -> transitions.
```

Pour le cercle 2, la question devient :

```text
comment decrire cette chaine sans creer de couche relationnelle superflue ?
```

### 6. Decision de recuperation

Decision :

```text
le bloc Saveur-Higgs cible est suffisamment remonte pour ouvrir le cercle 2.
```

Condition :

```text
la reprise devra traduire les anciens termes en roles positifs et ne pas
restaurer les etiquettes historiques comme rangs de classement.
```
