### 1. Fonction de CP-1

CP-1 prend en charge le point que le cycle Saveur-Higgs laissait en
amont :

```text
comment le regime electrofaible brise devient-il disponible pour
l'architecture Saveur-Higgs ?
```

La reponse de synthese est :

```text
le seuil BEH rend disponible un regime dans lequel `v` fonctionne
comme noyau d'echelle, les masses faibles comme temoins
phenomenologiques et les Yukawa comme relais vers la differenciation
fermionique.
```

Le cycle porte donc sur :

```text
le seuil electroweak comme architecture d'entree.
```

### 2. Decision centrale

Decision de rang :

```text
CP-1 = cycle de constance de seuil.
```

La constance de seuil se definit ici comme :

```text
un mode de stabilisation dans lequel une grandeur d'echelle rend un
regime physiquement lisible, coordonne des effets mesurables et
ouvre une architecture ulterieure.
```

Dans CP-1 :

```text
`v` = noyau d'echelle ;
`G_F` = route d'acces basse energie ;
`M_W`, `M_Z` = temoins faibles ;
Yukawa = relais Saveur-Higgs ;
brisure = nom operatoire du seuil.
```

| CP-2 | le secteur scalaire reste en reserve | CP-1 conserve son objet de seuil |

```text

`v` + masses faibles + `G_F` + relais Saveur-Higgs.
```
