# Synthèse de phase 1 — fixité électrofaible dynamique

## Statut

Cette synthèse clôt la phase de cadrage du cycle.

Elle ne conclut pas qu'une variation électrofaible réelle a eu lieu. Elle établit les conditions minimales permettant de poser cette hypothèse sans confondre plusieurs phénomènes distincts.

## 1. Déplacement produit

Le point de départ était :

```text
les paramètres électrofaibles pourraient-ils prendre d'autres valeurs ?
```

Le cycle ajoute :

```text
pourraient-ils évoluer dans un même univers ?
par quelle dynamique ?
comment deviendraient-ils presque fixes ?
```

Le problème passe d'un espace statique de mondes possibles à une famille de trajectoires cosmologiques.

## 2. Résultat principal

> Une variation électrofaible intra-univers est théoriquement formulable, mais elle exige généralement une extension dynamique du Modèle standard et ne peut être représentée par une simple fonction arbitraire `G_F(t)`.

Une théorie doit identifier :

- le porteur de la variation ;
- les fonctions de couplage ;
- les équations d'évolution ;
- les corrélations entre masses, couplages et taux ;
- le mécanisme de stabilisation ;
- les signatures observationnelles.

## 3. Quatre objets à ne pas confondre

| Objet | Variation possible | Signification |
|---|---|---|
| `g`, `g'` | RG ou fonction d'un champ | intensité des interactions de jauge |
| `v` | thermique, dérivée ou dynamique dans une extension | échelle du régime brisé |
| `Y_f` | RG ou fonction d'un champ | masses, textures et saveur |
| `G_F` | coefficient effectif dérivé | description basse énergie des processus faibles |

L'expression « force faible variable » est donc insuffisante.

## 4. Quatre sens du mot variation

Le cycle distingue :

```text
1. dépendance RG ;
2. dépendance thermique ;
3. variation dynamique spatio-temporelle ;
4. variation inférentielle.
```

Le gain conceptuel est décisif :

```text
même symbole variable
≠
même phénomène physique
```

## 5. Constitution dynamique d'une fixité

Le cycle introduit le problème suivant :

> Comment une grandeur variable devient-elle quasi stationnaire relativement à un régime, une durée et une tolérance ?

Les mécanismes candidats sont :

- minimum d'un potentiel ;
- friction cosmologique ;
- attracteur ;
- moindre couplage ;
- masse tardive ;
- écrantage ;
- transition de phase ;
- couplage au Higgs ou à la courbure ;
- découplage tardif.

Aucun de ces mots ne constitue seul une explication.

## 6. Critère explicatif

Une dynamique améliore l'explication seulement si elle produit :

```text
large bassin de conditions initiales
-> convergence des trajectoires
-> valeur tardive robuste
-> variation résiduelle compatible avec les observations
```

et si le coût en paramètres n'est pas supérieur au réglage initial.

## 7. Profil observationnel

Les contraintes imposent un profil temporel fortement structuré.

### Très tôt

Une variation importante peut être concevable avant ou autour du changement de régime électrofaible.

### BBN

Les calculs récents limitent typiquement la différence relative de l'échelle faible à un niveau percentuel, sous hypothèses de corrélation déterminées.

### Recombinaison

Le CMB limite les variations de la masse de l'électron au niveau approximatif du pour cent et celles de `alpha` plus fortement, selon les analyses et les dégénérescences cosmologiques.

### Univers tardif

Horloges, spectroscopie, équivalence et cinquièmes forces exigent une variation résiduelle extrêmement faible ou écrantée.

## 8. Premier modèle test

Le cycle retient provisoirement :

```text
scalaire phi
+ portail lambda_Hphi phi² H†H
-> terme quadratique du Higgs dépendant de phi
-> v(phi)
-> masses et G_F corrélés
```

Ce modèle est choisi pour sa lisibilité, non pour sa supériorité théorique.

Il permet de tester directement :

- bassin d'attraction ;
- époque de stabilisation ;
- BBN ;
- dérive tardive ;
- mélange scalaire ;
- migration du réglage.

## 9. Relation au weakless

Les modèles weakless demandaient :

```text
quelles histoires sont possibles sous une force faible globalement différente ?
```

Le nouveau cycle demande :

```text
quelles histoires du secteur faible lui-même sont dynamiquement possibles ?
```

Le weakless compare des mondes paramétriques.

Le présent cycle compare des trajectoires au sein d'une théorie dynamique.

Les deux enquêtes sont complémentaires mais non substituables.

## 10. Relation à la contingence canalisée

Le cycle donne une formulation plus précise :

> Une contingence est canalisée lorsque la dynamique réduit l'ensemble des trajectoires tardives accessibles sans imposer directement leur résultat comme condition initiale.

Trois degrés sont distingués :

### Canalisation faible

Le champ ralentit mais conserve la mémoire de sa valeur initiale.

### Canalisation forte

Un attracteur efface largement cette mémoire.

### Fixité constituée

Le système atteint un régime où la variation résiduelle reste sous la tolérance empirique pendant une durée spécifiée.

## 11. Risque central

Le risque principal est la migration du réglage :

```text
valeur électrofaible choisie
-> potentiel du champ choisi
-> portail choisi
-> conditions initiales choisies
```

Le cycle ne sera concluant que s'il peut quantifier cette migration.

## 12. Décision de phase 1

```text
Phase 1 : concluante comme cadrage.
```

Elle établit :

1. que la question est physiquement formulable ;
2. qu'elle dépasse le weakless classique ;
3. qu'elle exige une extension dynamique ;
4. que les contraintes observationnelles sont suffisamment fortes pour rendre le test discriminant ;
5. qu'un modèle minimal calculable peut être construit ;
6. que le résultat peut être négatif.

## 13. Phase 2

La phase 2 devra produire un dossier calculatoire minimal.

### Étape 1

Fixer les conventions du potentiel et les paramètres sans ambiguïté dimensionnelle.

### Étape 2

Résoudre la dynamique homogène de `phi` sur un fond cosmologique simplifié.

### Étape 3

Calculer `v(t)` et `G_F(t)`.

### Étape 4

Comparer plusieurs ensembles de conditions initiales pour tester l'attracteur.

### Étape 5

Imposer une borne BBN simplifiée, puis une borne tardive.

### Étape 6

Comparer le volume de paramètres viable avec celui du modèle sans dynamique.

### Étape 7

Décider :

```text
dissolution
réduction
déplacement
aggravation
ou indétermination
```

## 14. Formule de clôture

> Le passage d'une constante supposée fixe à une grandeur dynamique n'explique rien par lui-même. Il devient explicatif seulement lorsqu'une histoire physique montre pourquoi des trajectoires diverses convergent vers une relation tardive stable, et pourquoi cette convergence résiste aux contraintes et aux corrections.
