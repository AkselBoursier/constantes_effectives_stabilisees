#### 5. Pourquoi commencer par (G_F)

La constante de Fermi est le cas le plus pur de ce cycle.

Elle n’est pas seulement une grandeur numérique. Elle condense, à basse énergie, l’effet d’une interaction faible médiée par des bosons massifs.

Dans le Modèle standard, (G_F) est déterminée avec une très grande précision à partir de la durée de vie du muon ; une valeur couramment utilisée est :

[

]

Le point important pour la carte n’est pas seulement cette précision. Le point important est le statut de (G_F) : c’est une constante de raccordement effectif.
