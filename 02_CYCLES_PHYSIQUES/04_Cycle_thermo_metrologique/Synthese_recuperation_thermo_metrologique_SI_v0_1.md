# Synthese de recuperation thermo-metrologique SI v0.1

## Bloc k_B, N_A, R et quatrieme cycle

### 1. Fonction

Cette note recupere le bloc DOCX thermo-metrologique avant l'ouverture du
cercle 2 des architectures actives.

Sources remontees :

```text
Source_docx_Fiche_k_B_v0_1.md
Source_docx_Fiche_N_A_v0_1.md
Source_docx_Fiche_R_v0_1.md
Source_docx_Synthese_quatrieme_cycle_v0_1.md
```

La fonction de la note n'est pas de refaire le cycle SI actif.

Elle fixe ce que les sources anciennes apportent au cercle 2 :

```text
la difference entre role physique, statut metrologique et mode de
stabilisation conventionnelle.
```

### 2. Recuperation principale

Le bloc confirme trois niveaux distincts.

| Grandeur | Recuperation utile | Statut pour la reprise |
|---|---|---|
| `k_B` | Passage temperature / energie / entropie | Constante definissante SI avec role physique irreductible |
| `N_A` | Pont entites microscopiques / quantite de matiere | Constante definissante SI, distincte de `k_B` |
| `R` | Composition molaire `R = N_A k_B` | Exactitude par composition, non source definissante autonome |
| Synthese quatrieme cycle | Compression empirique / physique / conventionnelle | Source de garde-fou methodologique |

Le point le plus important est positif :

```text
l'exactitude numerique dans le SI stabilise un mode d'acces et de
definition ; elle n'annule pas la fonction physique de la grandeur.
```

### 3. Distinctions a conserver

Le bloc impose au cercle 2 de distinguer :

```text
1. constante definissante primaire ;
2. grandeur exacte par composition ;
3. role physique local ;
4. mode de stabilisation conventionnelle ;
5. histoire empirique qui rend la convention possible.
```

Cette distinction evite deux reductions.

Premiere reduction :

```text
dire que `k_B`, `N_A` et `R` sont simplement "conventionnels".
```

Deuxieme reduction :

```text
traiter `R = N_A k_B` comme une simple relation formelle.
```

La lecture robuste est :

```text
`k_B` et `N_A` sont des constantes definissantes ;
`R` est une grandeur composee exacte ;
les trois portent des fonctions physiques differentes.
```

### 4. Effet sur l'architecture SI

Le bloc renforce l'architecture metrologique SI active.

Il precise que l'architecture SI ne doit pas etre decrite comme une simple
liste de constantes exactes.

Elle articule :

```text
definition d'unites,
fermeture conventionnelle,
heritage empirique,
roles physiques conserves,
et compositions exactes internes.
```

Pour le cercle 2, la question devient :

```text
comment decrire une architecture definitionnelle sans absorber toute la
physique dans la convention ?
```

### 5. Decision de recuperation

Decision :

```text
le bloc thermo-metrologique SI est suffisamment remonte pour ouvrir le
cercle 2.
```

Condition :

```text
le cercle 2 devra conserver la separation entre exactitude definissante,
exactitude derivee, role physique et regime d'acces metrologique.
```
