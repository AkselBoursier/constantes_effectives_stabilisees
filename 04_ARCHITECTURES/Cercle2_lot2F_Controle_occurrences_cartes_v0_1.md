# Cercle 2 lot 2F - Controle d'occurrences et cartes v0.1

## Controle de cloture du cercle 2 architectural

### 1. Fonction

Cette note produit le lot 2F du cercle 2.

Elle controle la reprise positive des architectures apres les lots 2A a 2E.

Appuis :

```text
04_ARCHITECTURES/Cercle2_architectures_actives_v0_1.md
04_ARCHITECTURES/Cercle2_lot2A_Saveur_Higgs_v0_1.md
04_ARCHITECTURES/Cercle2_lot2B_Metrologique_SI_v0_1.md
04_ARCHITECTURES/Cercle2_lot2C_Effective_basse_energie_v0_1.md
04_ARCHITECTURES/Cercle2_lot2D_Cosmologique_v0_1.md
04_ARCHITECTURES/Cercle2_lot2E_Synthese_transverse_architectures_v0_1.md
```

Le controle porte sur trois points :

```text
1. les occurrences des idiomes surveilles ;
2. leur statut methodologique ;
3. les raccords documentaires necessaires avant la carte consolidee suivante.
```

### 3. Statut des idiomes surveilles

| Idiome | Statut apres cercle 2 | Decision |
|---|---|---|
| `famille` | Terme conserve pour la sortie taxonomique controlee et les garde-fous historiques | Ne plus l'utiliser comme ouverture architecturale |
| `relation` | Terme conserve seulement pour les cas qualifies : composition, rapport, equation d'etat, orientation, spectre | Remplacer toute case large par une forme locale nommee |
| `inter-familles` | Terme genealogique encore utile pour retrouver l'ancien niveau | Preferer `architecture active` ou `solidarisation de fonctions` dans les nouvelles syntheses |
| `architecture` | Terme directeur stabilise | Le garder, mais toujours avec regime, acces, trajectoire et limite |

Decision :

```text
le probleme n'est pas l'occurrence brute ;
le probleme est l'occurrence qui gouverne la phrase sans statut methodologique.
```

### 4. Zones propres

Les lots 2A a 2D remplissent la fonction attendue.

Ils presentent chaque architecture par ce qu'elle fait :

```text
Saveur-Higgs : constitution.
Metrologique SI : definition et realisation.
Effective basse energie : validite situee.
Cosmologique : inference et reconstruction.
```

Les occurrences surveillees y apparaissent principalement dans :

```text
1. les sections de formulations a eviter ;
2. les rappels de garde-fou ;
3. les formulations de statut.
```

Ces zones sont acceptables.

### 5. Zones a ne pas reprendre mecaniquement

Les notes anciennes ne doivent pas etre purifiees ligne a ligne maintenant.

Documents concernes :

```text
04_ARCHITECTURES/Architecture_saveur_Higgs_notes.md
04_ARCHITECTURES/Architecture_metrologique_SI_notes.md
04_ARCHITECTURES/Architecture_effective_basse_energie_notes.md
04_ARCHITECTURES/Architecture_cosmologique_notes.md
05_CARTES_ET_SYNTHESES/Synthese_architectures_inter_familles_v1_0.md
```

Raison :

```text
ces documents conservent la genealogie du travail.
```

Ils seront depasses par :

```text
1. la carte consolidee suivante ;
2. une note methodologique courte ;
3. la matrice des acquis ;
4. les reprises locales futures seulement si un conflit apparait.
```

### 6. Regle de controle pour les cartes suivantes

Toute carte consolidee produite apres le cercle 2 doit suivre cette regle :

```text
ouvrir par les modes de stabilisation,
puis seulement ensuite mentionner les controles de rang.
```

Ordre recommande :

```text
1. stabilisations effectives situees ;
2. modes architecturaux ;
3. regimes d'acces ;
4. temporalites ;
5. limites conservees ;
6. sortie taxonomique controlee ;
7. anciens idiomes en statut historique.
```

### 7. Raccords documentaires a effectuer

Raccords necessaires :

```text
README.md
05_CARTES_ET_SYNTHESES/Index_raisonne_du_corpus_v1_0.md
05_CARTES_ET_SYNTHESES/Carte_consolidee_v1_2_refonte.md
04_ARCHITECTURES/Cercle2_architectures_actives_v0_1.md
```

Ces raccords ne doivent pas produire une nouvelle these.

Ils doivent simplement rendre le cercle 2 retrouvable et indiquer la suite.

### 8. Decision de cloture

Decision :

```text
le cercle 2 architectural est suffisamment stabilise pour etre cloture
comme sequence de reprise positive.
```

Cette cloture ne signifie pas que les notes anciennes sont parfaites.

Elle signifie que le corpus dispose maintenant d'une couche positive active
sur laquelle construire la carte consolidee suivante.

### 9. Suite logique

La prochaine etape n'est plus un nouveau lot du cercle 2.

Elle devient :

```text
Carte consolidee v1.3 post-cercle 2.
```

Fonction de cette carte :

```text
stabiliser l'etat du corpus apres les reprises positives,
en preparant la note methodologique courte et la matrice des acquis.
```

### 10. Statut du lot 2F

Statut :

```text
lot 2F produit.
```

Formule de cloture :

```text
Les anciens idiomes restent lisibles comme traces de construction ;
les documents actifs doivent maintenant porter la formulation positive des
regimes, acces, trajectoires, solidarites et limites.
```
