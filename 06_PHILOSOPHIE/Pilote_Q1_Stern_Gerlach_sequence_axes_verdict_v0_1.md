# Chantier Q (catégories, accès et inférences) — Séquences d'axes : verdict v0.1

## Statut

```text
statut : verdict du second test ;
appui : Pilote_Q1_Stern_Gerlach_sequence_axes_protocole_v0_1.md ;
question : que change l'enchaînement d'analyseurs selon des axes identiques ou incompatibles ?
```

## 1. Fonctions différenciées

| Opération | Fonction | Inférence interdite |
|---|---|---|
| séparation | corréler spin et position | un résultat macroscopique défini existe déjà |
| détection | produire une trace | la trace est identique à la propriété |
| sélection | transmettre une sortie | l'opération est une lecture passive |
| préparation | fournir l'entrée suivante | l'état assigné impose son ontologie |
| recombinaison | tester ou restaurer une cohérence | la réversibilité pratique est parfaite |
| décohérence | rendre une phase localement inaccessible | le résultat unique est expliqué |
| changement d'axe | changer l'observable analysée | un vecteur classique complet est simplement tourné |

## 2. Résultat principal

Deux chaînes différentes peuvent donner la même distribution finale selon `z` :

```text
A. sélectionner +x, puis analyser z ;
B. séparer en ±x, perdre la cohérence ou ignorer le chemin, puis analyser z.
```

Mais elles ne préparent pas le même état intermédiaire :

```text
A : état pur |+x> ;
B : mélange 1/2 |+x><+x| + 1/2 |-x><-x|.
```

Une seule statistique terminale sous-détermine donc l'histoire de préparation. Il faut distinguer :

```text
équivalence statistique locale ;
équivalence d'état ;
équivalence de préparation ;
équivalence pour les opérations futures.
```

## 3. Catégories requalifiées

### Mesure

Le mot peut désigner une pré-mesure cohérente, une analyse à deux sorties, une mesure sélective, une mesure non sélective, une détection ou un conditionnement. Il est trop grossier sans précision opératoire.

### Préparation

Une sortie sélectionnée d'un premier analyseur prépare l'entrée du suivant. La distinction préparation/mesure dépend donc de la place du dispositif dans la séquence.

### Perturbation

Il faut séparer interaction physique, transformation unitaire, blocage d'une branche, conditionnement, décohérence et mise à jour descriptive.

### Résultat

Une trace finale ne reconstruit pas nécessairement l'état ni les opérations qui l'ont précédée.

## 4. Accès distribués

```text
matériel : orientation, séparation, filtrage, recombinaison, détection ;
opératoire : choix des axes et des sorties conservées ;
représentationnel : base, projecteurs, état pur ou densité ;
inférentiel : passage des fréquences à la préparation ou à la composante ;
accès à la cohérence : opération sensible à une phase relative.
```

L'absence d'accès à une phase dans une lecture donnée ne prouve pas son inexistence dans l'état composé.

## 5. Gain non trivial

Le second test interdit désormais des passages précis :

```text
statistique terminale -> préparation unique ;
séparation spatiale -> mélange classique ;
sélection -> simple lecture ;
répétabilité selon un axe -> possession classique de toutes les composantes ;
absence d'interférence observée -> disparition ontologique de la cohérence.
```

## 6. Limite

La théorie des instruments quantiques possède déjà les distinctions entre opérations sélectives, non sélectives et préparations conditionnelles. Le pilote ne les découvre pas.

Sa possibilité propre reste de croiser ces distinctions formelles avec les fonctions matérielles, inférentielles, communicationnelles, pédagogiques et ontologiques des mêmes catégories, puis de tester cette matrice sur plusieurs expériences.

## 7. Maturité après le second test

Les niveaux `C0–C4` désignent ici la maturité de la contribution.

```text
C0 : établi ;
C1 : renforcé ;
C2 : plausible ;
C3 : non établi ;
C4 : non revendiqué.
```

## 8. Suite recommandée

Une troisième variante de Stern–Gerlach apporterait peu. Le prochain test du
chantier Q devait changer de type d'accès : expérience d'information de chemin
et d'interférence, avec comparaison de `chemin`, `trace`, `visibilité`,
`information`, `enregistrement`, `décohérence` et `événement`.

> Le second test montre qu'une catégorie classique distribue plusieurs fonctions dans une chaîne expérimentale, et qu'une même donnée finale peut perdre l'information permettant de distinguer plusieurs chaînes. La question porte sur ce que chaque accès autorise à attribuer, non sur une opposition globale entre langage classique et réalité quantique.
