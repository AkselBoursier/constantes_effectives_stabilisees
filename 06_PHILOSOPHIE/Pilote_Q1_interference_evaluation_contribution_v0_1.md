# Chantier Q (catégories, accès et inférences) — Interférence : évaluation de contribution v0.1

## Verdict synthétique

```text
nouveauté physique revendiquée : aucune ;
conversation engagée : établie ;
recouvrement reconnu : établi en première passe ;
différenciation de contribution : plausible ;
limites conservées : établies ;
transférabilité générale de la matrice : non établie.
```

## Conversations réelles

Le test rejoint quatre conversations déjà constituées :

```text
complémentarité quantitative entre visibilité et distinguabilité ;
rôle de l’intrication et de la décohérence dans la perte des franges marginales ;
interprétation des effaceurs quantiques et du choix retardé ;
statut épistémique de l’information, du résultat et de l’enregistrement.
```

## Recouvrement honnête

Les distinctions physiques centrales sont déjà disponibles :

- `D² + V² <= 1` dans la littérature sur la dualité ;
- différence entre corrélation de voie et lecture effective ;
- différence entre distribution marginale et distributions conditionnelles ;
- frange et antifrange dont la somme est sans franges ;
- absence de rétroaction sur les événements déjà enregistrés.

Le projet ne doit donc pas présenter ces distinctions comme découvertes.

## Différenciation proposée

La contribution candidate est une matrice à trois dimensions :

```text
catégorie employée
x statut de l’accès
x inférence autorisée ou interdite.
```

Exemple pour « information » :

```text
encodée -> existence d’une corrélation ;
accessible -> existence d’une opération discriminante ;
lue -> production d’un résultat ;
enregistrée -> disponibilité publique d’une trace ;
utilisée -> conditionnement effectif d’une attribution.
```

Cette coordination rend plus précises deux formulations fréquentes :

```text
« connaître la voie supprime l’interférence » ;
« l’effaceur fait réapparaître l’interférence ».
```

Elles doivent préciser qui ou quoi « connaît », le statut de la corrélation, le niveau statistique considéré et l’opération de conditionnement.

## Limites

La matrice :

```text
ne remplace pas la théorie quantique de l’information ;
ne tranche pas entre interprétations ;
ne démontre pas une trajectoire réelle ;
ne transforme pas une clarification pédagogique en résultat physique ;
ne prétend pas que tous les usages d’« information » se réduisent à cinq statuts.
```

## Incidence sur le chantier Q

Le troisième test modifie le verdict précédent : l’absence de nouveauté ne
clôt pas le chantier Q. La matrice possède une contribution clarificatrice
plausible parce qu’elle coordonne des distinctions techniques, expérimentales,
inférentielles et communicationnelles souvent exposées séparément.

Les niveaux `C0–C4` (maturité de la contribution) donnent le verdict suivant :

```text
C0 — correction et documentation : établi ;
C1 — coordination de couches : établi ;
C2 — contribution différenciée : plausible, à éprouver bibliographiquement ;
C3 — protocole transférable : non établi ;
C4 — nouveauté substantielle : non recherchée.
```

## Suite

Avant un nouveau cas, il faut confronter la matrice aux travaux qui traitent déjà explicitement :

- les niveaux ou usages de l’information quantique ;
- l’analyse pragmatique des expériences d’effacement ;
- la distinction entre information disponible et information enregistrée ;
- les migrations de catégories entre formalisme, expérience et discours.

La question n’est plus de savoir si le terrain est occupé, mais si cette coordination possède une forme suffisamment précise et utile pour y contribuer.
