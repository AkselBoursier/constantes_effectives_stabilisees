### 1. Fonction

Le premier cycle a fait dialoguer le corpus avec sept configurations
historiques. Cette synthese repond a une question simple :

```text
qu'est-ce que les sept fiches etablissent ensemble
qu'aucune n'etablit seule ?
```

Huit degagements transversaux repondent. Ils sont numerotes D1 a D8
pour les distinguer des degagements theoriques T0-T10, qu'ils
nourrissent sans les remplacer.

### 2. D1 - La genealogie des querelles d'acces

```text
trois querelles, un meme defaut, trois objets :
1908, Mach contre Planck : les constantes (S07) ;
1922, Bergson contre Einstein : le temps (S01) ;
1922, Whitehead contre Einstein : la mesure (S01).

le defaut commun : convertir l'acces en essence, par exces (l'acces
epuise le statut : Mach, Bergson pour le vecu) ou par defaut
(le statut expulse l'acces : Planck, Einstein pour le temps).

la forme positive de N6 - l'acces comme element de qualification du
statut - est la resolution que les trois querelles n'avaient pas.
Le corpus n'invente pas le probleme : il herite d'un siecle de
querelles mal outillees, et c'est un argument pour les papiers.
```

### 3. D2 - Une typologie des erreurs de rang (point de decision)

Les fiches ont diagnostique quatre erreurs, deux a deux symetriques :

| Erreur | Forme | Cas historique |
|---|---|---|
| Aplatissement | Traiter des statuts heterogenes comme homogenes | Dirac : alpha_G (rapport), t (indexical), N (compte reconstruit) fondus en "constantes cosmologiques" (S02) |
| Inversion statut-valeur | Lire un changement de statut comme un evenement physique | Le theoricien du futur devant le k_B de 2019 (S03) |
| Acces-essence par exces | Absorber le statut dans l'acces | Mach, Bergson (S07, S01) |
| Acces-essence par defaut | Expulser l'acces du statut | Planck, Einstein (S07, S01) |

```text
proposition (a valider) : cette typologie merite d'entrer dans
l'outillage methodologique comme grille de diagnostic - elle a
fonctionne sur sept cas historiques sans exception rencontree.
```

### 6. D5 - La solidarite est mesurable : trois instruments convergents

```text
1. le test de retrait mesure la solidarite par soustraction et
   revele des degres (holisme fort constitutif, faible detachable,
   inferentiel fort : S05) ;
2. l'experience naturelle du regime gris montre ou la solidarite
   reside quand droit et fait se separent : dans les constantes et
   leurs effets, non dans les codes (S04) ;
3. la covariance CODATA chiffre la solidarite par propagation,
   paire par paire (S06 ; chantier candidat).

ensemble, ils transforment T5 : l'architecture cesse d'etre une
description pour devenir un objet de mesure. Le holisme n'est pas
un camp ; c'est une grandeur (S05).
```

### 7. D6 - Trois pieces historiques pour H1

L'hypothese H1 (couplage formation epistemique / mode de
stabilisation) recoit trois pieces avant meme son test formel :

```text
1. 1900 : Planck revendique la concordance de N_A radiatif avec les
   routes cinetiques - convergence multi-acces comme argument, chez
   l'acteur meme du realisme des constantes (S07) ;
2. 1990 : l'adoption de K_J-90 et R_K-90 est gatee par les tests
   d'universalite - condition de convergence avant convention
   sectorielle (S04) ;
3. 2017-2019 : les conditions d'entree de l'absorption systemique
   sont ecrites et votees - le couplage devient institutionnel (S03).

le meme patron a trois echelles et trois epoques. Le test formel
(plan d'arene, axe 5) part avec un prejuge favorable documente,
ce qui oblige a chercher les contre-exemples d'autant plus durement.
```

### 10. Garde de la synthese

```text
le risque propre du cycle est l'histoire interessee : lire chaque
episode comme confirmation de la grille. Trois protections restent
actives :
1. les verdicts des fiches sont factuels et sources, la ou les
   lectures retrospectives sont marquees comme telles ;
2. les sources primaires listees "a verifier" gardent ce statut
   jusqu'a lecture ; aucun degagement D ne s'affirme au-dela ;
3. les degagements D restent des consolidations de branche : leur
   promotion vers les degagements T ou la methode passe par le
   circuit de validation.
```
