# Protocole candidat — transfert inférentiel v0.1

## Statut

```text
statut : hypothèse méthodologique à comparer aux cadres existants ;
originalité : non établie ;
fonction : décider si un résidu subsiste après l'audit négatif de Q1.
```

## Question

Lorsqu'une même catégorie passe d'une couche scientifique à une autre, quelles inférences restent autorisées et quel pont justifie leur maintien ?

## Fiche minimale

Pour chaque occurrence d'une catégorie `C`, relever :

```text
1. couche source : matérielle, opératoire, formelle, inférentielle,
   communicationnelle, pédagogique ou ontologique ;
2. fonction locale de C ;
3. inférence I autorisée dans cette couche ;
4. couche cible vers laquelle I est transportée ;
5. pont W : théorème, modèle, calibration, invariance empirique,
   convention, argument interprétatif ou absence de pont ;
6. condition d'échec de W ;
7. conséquence du transfert.
```

Forme schématique :

```text
(C, couche_source, fonction_source, I)
        -- W -->
(C, couche_cible, fonction_cible, I').
```

## Calibration Stern–Gerlach

### « Deux trajectoires »

```text
source : schéma spatial ou approximation semi-classique ;
inférence locale : deux régions de propagation ;
cible : histoire microscopique individuelle ;
pont requis : modèle de trajectoire déterminé ;
verdict : aucun pont général fourni par le schéma seul.
```

### « L'appareil mesure S_z »

```text
source : opérateur du modèle idéal ;
cible : appareil réel ;
pont requis : régime, approximations, calibration et résolution ;
verdict : transfert légitime seulement sous conditions documentées.
```

### « +z est une propriété préexistante »

```text
source : étiquette d'issue et état conditionnel ;
cible : attribution ontologique antérieure ;
pont requis : interprétation supplémentaire ;
verdict : le formalisme opérationnel seul ne fournit pas ce pont.
```

## Voisinages à contrôler

```text
conception inférentielle de la représentation ;
modèles comme médiateurs ou artefacts épistémiques ;
inférentialisme et rôle conceptuel ;
modélisation des instruments et de l'observateur ;
analyse des idéalisations ;
changement conceptuel, polysémie et objets-frontières.
```

## Seuil de maintien

Le protocole ne sera conservé comme résultat de recherche que s'il :

```text
A. distingue une erreur non déjà classée par ses voisins ;
B. donne des verdicts suffisamment convergents entre analystes ;
C. produit au moins une correction réelle et non lexicale ;
D. possède une condition d'échec explicite.
```

Sinon, il sera reclassé comme outil pédagogique et documentaire.
