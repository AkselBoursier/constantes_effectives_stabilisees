# Pilote Q1 — Stern–Gerlach

## Statut

```text
branche : agent/pilote-q1-stern-gerlach ;
base : agent/cartographie-questions-contemporaines ;
statut : deux tests produits ;
fonction : tester une typologie fonctionnelle des catégories classiques ;
interdit : transformer Stern–Gerlach en illustration d'une doctrine préalable.
```

## Question

> Quelles fonctions distinctes les catégories classiques exercent-elles dans la préparation, le contrôle, l'enregistrement, l'inférence et la communication d'une expérience de Stern–Gerlach, et quand deviennent-elles trompeuses comme attributions ontologiques ?

## Distinction préalable

Le nom `Stern–Gerlach` recouvre au moins trois objets :

```text
H — expérience historique de 1922 : faisceau d'atomes d'argent et dépôt cumulatif ;
I — modèle quantique idéalisé : couplage spin–position et paquets d'onde ;
M — dispositifs modernes ou séquentiels : préparation, analyse et détection résolue.
```

Les mots `impact`, `résultat`, `trajectoire`, `particule`, `mesure` et `deux issues` changent de fonction entre H, I et M.

## Résultats acquis

```text
1. expérience historique, modèle idéal et dispositif moderne ne donnent pas les mêmes objets de résultat ;
2. axe, trajectoire, trace et propriété possèdent des profils fonctionnels distincts ;
3. un même appareil peut analyser, sélectionner ou préparer selon l'usage de ses sorties ;
4. séparation cohérente, mélange et détection ne sont pas équivalents ;
5. une statistique terminale peut sous-déterminer la préparation et la chaîne d'accès.
```

## Critère de réussite

Le pilote réussit seulement si la matrice produit des profils réellement différents et interdit des inférences précises. Il échoue s'il conclut seulement que « le langage classique est nécessaire ».

## Portée interdite

Ce pilote ne décide pas :

```text
quelle interprétation de la mécanique quantique est vraie ;
si la mesure crée ou révèle une propriété en général ;
si une coupure unique explique l'expérience ;
si les catégories classiques sont fondamentales ;
si Stern–Gerlach résout le problème du résultat défini.
```
