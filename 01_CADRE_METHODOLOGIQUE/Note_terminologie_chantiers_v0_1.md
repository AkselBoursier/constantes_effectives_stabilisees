# Note terminologique - codes de chantier v0.1

## Seuil electrofaible, chantiers physiques et jonctions

### 0. Statut documentaire

```text
statut : note terminologique legere ;
fonction : reserver les codes internes au reperage documentaire ;
portee : fichiers actifs du seuil electrofaible, chantiers physiques,
jonctions philosophie/physique et index.
```

Cette note ne change pas le rang des chantiers.

Elle fixe seulement une regle de lecture :

```text
le code sert a retrouver un dossier ;
l'intitule descriptif porte le sens dans la prose.
```

### 1. Regle generale

Dans les paragraphes explicatifs, les titres internes et les formules de
synthese, employer d'abord l'intitule descriptif.

Conserver le code dans trois cas :

```text
1. titre de fichier ;
2. statut documentaire ;
3. tableau de correspondance ou renvoi d'index.
```

### 4. Limite de la passe

La relecture terminologique ne doit pas effacer l'historique du corpus.

Les anciens documents peuvent conserver des codes lorsqu'ils servent a
retracer une decision.

Dans les documents actifs, la prose doit rester lisible sans connaitre
les codes.

### 5. Formule de cloture

```text
Un code de chantier aide a naviguer ;
il ne doit pas devenir le nom conceptuel de l'objet.
```
