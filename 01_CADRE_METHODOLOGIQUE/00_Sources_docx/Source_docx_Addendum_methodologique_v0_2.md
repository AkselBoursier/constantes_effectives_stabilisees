### Addendum méthodologique v0.2

#### Compression taxonomique, régimes d’accès et constantes fixées par convention

#### 1. Fonction de l’addendum

Cet addendum corrige trois fragilités de la méthode v0.1 avant l’ouverture du quatrième cycle.

Première fragilité : la typologie risque de se multiplier trop vite. Si chaque constante étudiée produit sa propre catégorie, la méthode décrit mais ne classe pas. Une taxonomie doit compresser : plusieurs grandeurs hétérogènes doivent pouvoir être rangées sous un même type.

Deuxième fragilité : le mot “régime” recouvre deux plans différents. Certains régimes sont physiques : domaine d’énergie, phase, symétrie brisée, régime confiné, régime cosmologique. D’autres sont des régimes d’accès : procédure de mesure, convention métrologique, schéma de renormalisation, ajustement de modèle, protocole expérimental.

Troisième fragilité : l’échelle de stabilisation empirique ne suffit pas. Certaines grandeurs sont fortement contraintes expérimentalement ; d’autres sont structurellement ouvertes ; d’autres encore sont fixées exactement par convention dans un système d’unités. Ces statuts ne relèvent pas du même axe.

#### 4. Révision de l’axe de stabilisation

La stabilisation ne doit plus être un axe unique. Elle doit être dédoublée.

##### 4.1 Stabilisation empirique

Cet axe indique le degré auquel la grandeur est mesurée, contrainte ou reconstruite.

| Niveau | Statut |
|---|---|
| Empiriquement forte | Mesure ou contrainte robuste |
| Partiellement stabilisée | Valeur générale connue, détail ouvert |
| En cours de stabilisation | Paramètre nécessaire mais valeur encore incertaine |
| Bornée | Limite supérieure ou inférieure sans mesure positive |
| Ouverte | Statut physique non tranché |

##### 4.2 Stabilisation conventionnelle

Cet axe indique si la grandeur reçoit une valeur exacte par définition dans un système d’unités.

| Niveau | Statut |
|---|---|
| Fixée par convention | Valeur exacte dans le SI ou un système d’unités donné |
| Déduite par convention | Valeur exacte par composition de constantes définissantes |
| Mesurée dans le système | Valeur déterminée expérimentalement dans les unités définies |
| Non concernée | Grandeur sans rôle direct dans la définition des unités |

Cela permet de traiter correctement les constantes définissantes du SI. Une constante comme (k_B) peut être physiquement centrale pour la thermodynamique statistique et conventionnellement exacte dans le SI. Ces deux statuts ne disent pas la même chose.

#### 5. Classe nouvelle : constante définissante d’un système d’unités

Le quatrième cycle doit introduire explicitement la catégorie :

constante définissante d’un système d’unités.

Définition provisoire :

Une constante définissante d’un système d’unités est une grandeur à laquelle un système métrologique attribue une valeur numérique exacte afin de définir une ou plusieurs unités. Sa stabilisation numérique relève alors d’une convention métrologique, non d’une mesure expérimentale continue.

Cette catégorie inclut notamment, dans le SI révisé :

[ h,e,k_B,N_A ]

Elle peut aussi inclure :

[ c,{},K{} ]

selon le périmètre adopté.

La conséquence méthodologique est décisive :

une valeur exacte n’est pas nécessairement une valeur mieux mesurée ; elle peut être une valeur stipulée.

#### 6. Cas dérivé : constante exacte par composition

La réforme du SI impose aussi une sous-catégorie :

constante exacte par composition.

Exemple :

[ R = N_A k_B ]

Puisque (N_A) et (k_B) ont des valeurs exactes dans le SI, (R) possède également une valeur exacte par composition. Son exactitude ne vient pas d’une mesure directe indépendante ; elle vient de la structure du système d’unités.

Définition provisoire :

Une constante exacte par composition est une grandeur dont la valeur numérique est exacte parce qu’elle résulte algébriquement de constantes définissantes exactes.

Cette catégorie sera testée par la fiche (R).

#### 7. Matrice courte révisée

La matrice minimale d’analyse devient :

| Critère | Question |
|---|---|
| Grandeur | De quoi parle-t-on exactement ? |
| Type logique | Valeur, rapport, couplage, échelle, angle, phase, coefficient, densité ? |
| Dimension | Dimensionnée ou sans dimension ? |
| Régime de définition physique | Dans quel domaine physique joue-t-elle un rôle ? |
| Régime d’accès épistémique | Par quelle procédure est-elle mesurée, fixée, reconstruite ou définie ? |
| Fonction | Que stabilise-t-elle ? |
| Dépendance d’échelle | Court-elle avec l’énergie ou dépend-elle d’un schéma ? |
| Stabilisation empirique | Mesurée, partielle, en cours, bornée, ouverte ? |
| Stabilisation conventionnelle | Fixée, dérivée, mesurée dans le système, non concernée ? |
| Limites | Qu’est-ce qu’elle ne dit pas ? |
| Famille fonctionnelle | Couplage, échelle, relation, raccordement, orientation, fond, seuil, convention ? |
| Sous-type éventuel | Type local si nécessaire seulement |

#### 8. Critère de succès du quatrième cycle

Le quatrième cycle ne doit pas seulement ajouter des fiches. Il doit tester la méthode révisée.

Le cycle sera réussi si :

il impose clairement la distinction entre stabilisation empirique et stabilisation conventionnelle ;

il permet de classer (k_B), (N_A) et (R) sans créer une nouvelle catégorie pour chaque grandeur ;

il montre que certaines constantes dimensionnées peuvent être physiquement centrales tout en étant numériquement exactes par convention ;

il oblige la taxonomie à mieux compresser.

Le cycle sera insuffisant si :

chaque cas produit une catégorie nouvelle ;

la distinction régime physique / régime d’accès reste décorative ;

la valeur exacte des constantes SI est traitée comme une mesure infiniment précise ;

(R) n’est pas reconnu comme cas dérivé par composition.

#### 9. Décision stratégique

Le quatrième cycle doit porter sur les constantes thermodynamiques et métrologiques :

[ k_B,N_A,R ]

Ce choix attaque la fragilité centrale de la méthode : la confusion possible entre constance physique, stabilisation empirique et fixation conventionnelle.

La séquence recommandée est :

[ k_B ]

comme constante de passage micro/macro et constante définissante du kelvin,

[ N_A ]

comme constante de comptage/mole et constante définissante de la quantité de matière,

[ R=N_Ak_B ]

comme constante exacte par composition.

Ce cycle doit être bref, fermé et méthodologiquement exigeant.
