# Note terminologique : constance et constante v0.1

## Codifier une distinction deja operative, sans renommer le cadre

### 0. Statut documentaire

```text
statut : point de decision terminologique, soumis a validation ;
origine : question posee en juillet 2026 lors de l'instruction du
cycle C5 - faut-il dire "constance" plutot que "constante", pour
inscrire la trajectoire dans le mot ?
rattachement : cadre methodologique (le vocabulaire engage tout le
corpus) ; s'articule a la reecriture positive v0.3 sans la modifier.
```

### 1. Le constat genealogique

La proposition semble neuve ; elle est en realite le plus vieux mot
du corpus qui remonte a la surface.

```text
le document fondateur s'intitule Regimes de constance (cadrage v0.2,
anterieure au PROJECT_BRIEF) ;
la these-noyau T0 dit : le statut de constance ;
T1 dit : les modes de constance ;
la fiche d'horizon anthropique ecrit : des constances effectives
stabilisees et situees ;
la situation 07 se clot sur : ce que vaut leur constance.
```

La distinction est donc deja operative, sans avoir ete codifiee :

```text
constante : la grandeur (alpha, G_F, k_B, H_0) ;
constance : le statut que le corpus etudie - ce qui se stabilise,
se maintient, se defait.
```

### 2. Analyse de la proposition forte (substitution generale)

Remplacer partout "constante" par "constance" :

```text
gains reels :
1. la trajectoire entre dans le mot : une constance s'acquiert, une
   constante semble donnee ;
2. protection anti-reification : le mot lui-meme decourage
   l'obstacle de substantialisation (situation 06) ;
3. economie : "constance effective" condense "constante effective
   stabilisee situee".

couts reels :
1. interface : la physique dit "constantes" ; des papiers qui n'en
   parleraient plus perdraient leurs lecteurs et leurs rapporteurs ;
2. ambiguite de langue : "constance" a un sens courant (la vertu de
   perseverance) qui parasiterait les titres ;
3. le geste, trop fort, dirait plus que la these : le corpus
   n'affirme pas que les constantes ne sont que des constances -
   N4 garde sa forme positive.

le diagnostic de l'auteur est retenu tel quel : simplification trop
forte, a mesurer.
```

### 3. Proposition mesuree : la regle des deux registres

```text
regle candidate (a valider) :

constante designe la grandeur, dans son usage physique et
metrologique - c'est le mot d'interface avec la litterature ;

constance designe le statut de cette grandeur - c'est le mot
technique de la these, deja porte par T0 et T1.

formule de cadrage : le corpus etudie la constance des constantes.

le nom du cadre ne change pas : "constantes effectives stabilisees"
nomme les objets ; la these porte sur leur constance.

note d'interface : l'anglais suit sans friction - the constancy of
constants est une tournure deja attestee dans la litterature sur la
variation des constantes.
```

Discipline d'usage associee :

```text
1. employer "constance" chaque fois que l'enonce porte sur le
   statut, la trajectoire, les modes ou les conditions ;
2. employer "constante" chaque fois que l'enonce porte sur la
   grandeur, sa valeur, sa mesure ;
3. reserver "constance" a cet usage technique : le mot ne sert pas
   de variante elegante de "constante".
```

### 4. Mise a l'epreuve prevue

```text
la regle se teste en usage sur les deux prochains documents de fond
(voisinages V1 et V2) avant decision definitive :
la distinction rend-elle les enonces plus nets, ou produit-elle des
lourdeurs et des ambiguites nouvelles ?
sortie : adoption, ajustement, ou maintien de l'usage actuel non
codifie.
```

### 5. Formule de cloture

```text
Le mot le plus ancien du corpus disait deja tout : des regimes de
constance. La grandeur s'appelle constante ; ce qu'elle accomplit
s'appelle constance ; et la these tient dans l'ecart entre les
deux mots.
```
