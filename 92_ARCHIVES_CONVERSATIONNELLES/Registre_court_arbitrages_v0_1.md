# Registre court des arbitrages v0.1

Ce registre conserve, en arrière-plan, les décisions qui changent réellement le
cours du projet après la dernière archive conversationnelle intégrale. Il rend
la transition reconstructible sans reproduire le clavardage et sans ajouter un
commentaire réflexif aux documents scientifiques.

## 0. Statut

```text
fonction : archiver silencieusement les arbitrages structurants ;
rang : archive de continuité, sans autorité scientifique ou doctrinale ;
début : 17 juillet 2026 ;
alimentation : courtes entrées ajoutées avec le travail concerné ;
visibilité : consultable dans le dépôt, absente du parcours scientifique actif ;
révision : le protocole peut être allégé ou interrompu sur décision explicite.
```

Le mot « silencieusement » signifie que l'entrée n'est pas présentée ni
discutée à chaque tour. L'archivage reste visible et vérifiable dans le dépôt.
Il peut être signalé lorsqu'il modifie lui-même le travail ou lorsque l'auteur
demande un état de provenance.

## 1. Seuil d'inscription

Une entrée est ajoutée lorsqu'au moins un des événements suivants se produit :

- une question de recherche, une priorité ou une portée est modifiée ;
- un chantier est ouvert, suspendu, fermé, réactivé ou placé en réserve ;
- le rang ou l'autorité documentaire d'un ensemble change ;
- une règle méthodologique est validée, corrigée ou abandonnée ;
- une non-décision détermine la séquence suivante ;
- une correction humaine change l'interprétation d'un résultat ou d'une
  opération déjà accomplie.

Restent hors registre les modifications de formulation sans effet de portée,
les opérations Git ordinaires, les validations d'exécution déjà prévues, les
explorations sans arbitrage et la répartition spéculative de la propriété
intellectuelle.

## 2. Forme des entrées

Chaque entrée tient en un micro-paragraphe naturel. Elle nomme :

1. le point de départ utile ;
2. l'arbitrage ou la non-décision ;
3. son effet sur le programme ou le corpus ;
4. ce qui reste hors décision ;
5. les documents qui permettent de reprendre le fil.

Une seule entrée peut réunir plusieurs tours lorsqu'ils forment le même
arbitrage. Aucune attribution fine des intuitions n'est ajoutée si elle n'est
pas nécessaire à la décision.

## 3. Arbitrages consignés

### 17 juillet 2026 — Réserve réflexive et continuité archivistique

Le chantier sur les rôles distribués a d'abord été placé entièrement en
réserve afin qu'il ne commande plus le travail scientifique. Une clarification
ultérieure conserve cette mise en réserve pour l'analyse réflexive, mais réactive
sa fonction la plus légère : extraire les arbitrages structurants dans le
présent registre sans les afficher à chaque étape. Le parcours actif reste
scientifique ; l'archivage silencieux devient une opération de continuité et ne
rouvre ni l'analyse des contributions ni la question de leur propriété. Les
traces de cette décision sont le [statut de la réserve
réflexive](../05_CARTES_ET_SYNTHESES/Index_raisonne_du_corpus_v1_3.md#31-outils-réflexifs-en-réserve)
et la [clôture du chantier dans la cartographie
conversationnelle](../05_CARTES_ET_SYNTHESES/Cartographie_selective_transitions_conversationnelles_v0_1.md#913-clôture-du-chantier-et-mise-en-réserve).
