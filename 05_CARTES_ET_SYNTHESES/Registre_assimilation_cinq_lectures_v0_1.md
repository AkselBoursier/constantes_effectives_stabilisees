## 1. Ancrer la philosophie dans la physique brute

### Diagnostic utile

- preuves physiques et résultats trop éloignés des entrées ;
- administration GitHub trop visible pour un lecteur externe ;
- besoin de confronter chaque proposition à son voisinage réel.

### Tri

```text
CONSERVER : accès rapide aux hypothèses, équations, résultats, contraintes et limites ;
CONSERVER : distinction scène intellectuelle / arrière-scène technique ;
ADAPTER : confrontation aux voisinages déterminés par la question locale ;
REJETER : convertir tout le projet dans la seule « langue native de la physique » ;
REJETER : traiter toute condition d'échec comme observable binaire.
```

### Destination

```text
navigation publique ; fiches de cas ; architecture des futurs livrables ; chantier A.
```

## 2. L'ordre heuristique des constantes fondamentales

### Diagnostic utile

- portes d'entrée concurrentes ;
- raccord insuffisant entre chantiers Q/M/A et lignées physiques ;
- accès trop étroitement placé en contrôle final.

### Tri

```text
CONSERVER : parcours narratif question -> ouverture -> cas -> résultat ;
CONSERVER : raccord question parente / cas / accès / sortie / échec / portée interdite ;
CONSERVER : accès présent aux étapes où il structure objet, opération ou preuve ;
ADAPTER : frise narrative sans effacer les bifurcations généalogiques ;
DIFFÉRER : formule « appareil co-déterminant l'observable » jusqu'au pilote Q1 ;
REJETER : seuil de décision binaire pour toute proposition philosophique.
```

### Destination

```text
README ; index ; méthode active ; pilote Q1 ; protocole A1.
```

## 3. La genèse des constantes de l'univers

### Valeur principale

Le document est peu fidèle comme résumé scientifique, mais très utile comme
symptôme de la doctrine radicale que le corpus peut sembler soutenir.

### Réservoir pédagogique

```text
différents types de glace : hétérogénéité sous un mot commun ;
toile d'araignée : stabilité relationnelle distribuée ;
question : comment soutenir l'historicité à partir de traces présentes ?
```

Chaque image exige un point d'arrêt : aucune n'établit une genèse dynamique commune.

### Contresens à enregistrer

```text
REJETER : toutes les constantes seraient des fixités dynamiques ;
REJETER : BEH expliquerait les valeurs des Yukawa ;
REJETER : la moindre énergie abolirait le fine-tuning ;
REJETER : le corpus établirait que les constantes évoluent encore ;
REJETER : l'accès serait seulement un contrôle a posteriori.
```

### Destination

```text
réserve pédagogique ; carte des contresens ; matériau du chantier A ; question M sur les traces de l'historicité.
```

## 4. Scénariser la genèse des constantes physiques

### Diagnostic utile

- compartimentation excessive ;
- transitions trop faibles ;
- méthode déclarée plutôt qu'incarnée par les cas.

### Tri

```text
CONSERVER : pont cas local -> distinction -> question générale -> portée limitée ;
CONSERVER : démontrer les distinctions sur des cas plutôt que les proclamer ;
RÉSERVOIR : image plans / coulage du béton, sous limites explicites ;
REJETER : constantes comme artefacts historiques temporaires ;
REJETER : cycle 10 comme métaphore ontologique centrale ;
REJETER : Stern–Gerlach comme genèse des constantes ;
REJETER : BEH comme preuve d'une coïncidence générale entre constitution et différenciation.
```

### Destination

```text
transitions ; pédagogie ; Q1 comme cas de description, non récit cosmique ; M1 comme test de portée explicative.
```

## 5. Traduire la mesure physique en philosophie

### Diagnostic utile

- liaison insuffisante entre résultat physique et portée philosophique ;
- preuves locales mal reliées aux questions générales ;
- jargon administratif trop présent dans l'exposition.

### Tri

```text
CONSERVER : résultat -> interprétation locale -> distinction -> portée bornée -> objection ;
CONSERVER : pour chaque cas, question instruite / gain local / portée interdite ;
CONSERVER : administration du dépôt reléguée à la provenance pour les textes publics ;
ADAPTER : accès comme dimension plurielle, non pont méthodologique unique ;
REJETER : traduction directe d'une mesure en conclusion philosophique.
```

### Destination

```text
méthode de transition ; Q1 ; M1 ; architecture publique ; chantier A.
```

## 6. Analogie de la toupie — correction d'auteur

L'analogie n'est conservée que pour un mouvement cognitif structurel :

```text
un comportement mobile ou irrégulier peut entrer dans un régime plus stable ;
une photographie prise dans une fenêtre donnée peut donner une impression de fixité ;
les échelles temporelles modifient la description pertinente.
```

Point d'arrêt absolu :

```text
aucune représentation de l'électron ;
aucune représentation du spin ;
aucune rotation microscopique classique ;
aucune loi générale de stabilisation des constantes.
```

Le passage exact consacré à la toupie n'apparaît pas dans les résumés extraits ;
son sens et ses limites sont donc enregistrés à partir de la correction explicite
de l'auteur.
