## 1. Principe

Le pan philosophique s'etend au-dela de l'echelle de la qualification.

La qualification reste l'axe le plus stabilise pour un premier livrable, mais
elle occupe seulement le rang d'entree actuellement le plus lisible.

Regle :

```text
avant de choisir un livrable, ouvrir les foyers philosophiques du projet,
extraire les axes epistemologiques depuis les cycles physiques, puis instruire
les echelles philosophiques ouvertes par le corpus et evaluer leur maturite
relative.
```

Cette instruction reste selective.

Elle vise :

```text
1. nommer les foyers philosophiques larges du projet ;
2. nommer les axes epistemologiques issus des cycles physiques ;
3. identifier les echelles philosophiques correspondantes ;
4. utiliser les situations comme resonances historiques et tests de contraste ;
5. distinguer acquis, promesses, fragilites et dettes de lecture ;
6. mesurer l'effet de chaque echelle sur A1, A2, A4 ou B ;
7. choisir ensuite le livrable le plus juste.
```

## 2. Echelles a instruire

| Echelle | Question directrice | Materiau disponible | Effet possible sur les livrables |
|---|---|---|---|
| Qualification | A quelles conditions une grandeur vaut-elle comme constante ? | T0, T3, T6, limites, acces, regimes | A2 probable |
| Acces et coordination | Comment une route d'acces stabilise-t-elle un statut ? | SI, coordination, Massimi, Tal, van Fraassen, Papale | A2 ou A1 |
| Realisme medie | Comment decrire des processus reels a travers des acces situes ? | note realiste mediee, SI, cosmologie, limites | A2, garde-fou general |
| Formation genetique | Comment une grandeur devient-elle constante ? | Q2, H1, expansion/acces, BEH, Mach/Planck | B possible |
| Modalite | Quel rapport entre necessite, contingence, invariance et stabilite ? | Dirac/Eddington, grands nombres, fine-tuning, constantes dimensionnelles | A3 ou B |
| Historicite | Comment une stabilisation transforme-t-elle le statut d'une grandeur ? | CODATA, regime gris, SI 2019, Bachelard | A1, A2, B |
| Metrologie | Comment definition, realisation et latitude se coordonnent-elles ? | SI, regime gris, critere de latitude | A1 fort |
| Architecture | Comment les dependances structurent-elles la robustesse ? | Duhem-Quine, test de retrait, cartes d'architecture | A2 ou A4 |
| Exportabilite | Comment le cadre discrimine-t-il hors terrain natif ? | tests C5, constante solaire, Michaelis, mathematiques | A4 |
| Constitution / seuil | Comment un mecanisme constitue-t-il une stabilisation ? | BEH, seuil electrofaible, masse, brisure | B ou dossier de jonction |
| Arene anthropique | Comment le corpus dialogue-t-il avec fine-tuning et selection ? | plan d'arene, constantes cosmologiques, modalite | horizon differe |

Cette table sert de carte d'attention.

## 3. Methode d'instruction

Chaque echelle recoit une fiche courte selon le meme schema :

```text
1. question directrice ;
2. situations disponibles ;
3. concepts utiles ;
4. apports pour le corpus ;
5. risques de formulation ;
6. dettes de lecture ;
7. effet sur les livrables ;
8. statut de maturite.
```

Statuts de maturite :

```text
P1 : acquis mobilisable ;
P2 : formule stable ;
P3 : instrument utile ;
P4 : hypothese candidate ;
P5 : programme ouvert ;
P6 : horizon de recherche.
```

## 5. Effet sur l'ordre des livrables

Avant instruction, l'hypothese de travail est :

```text
A2 est le meilleur candidat theorique ;
A1 est le meilleur candidat tactique ;
A4 est le meilleur candidat methodologique ;
B est le meilleur candidat original de deuxieme temps.
```

Apres instruction, cette hierarchie peut changer.

Exemples :

```text
si la metrologie et la latitude dominent, A1 passe devant ;
si coordination et realisme medie tiennent mieux, A2 reste devant ;
si les tests C5 donnent la meilleure originalite sobre, A4 devient attractif ;
si formation genetique et modalite montent vite, B devient le chantier fort,
mais avec un delai de consolidation.
```

## 7. Formule de cloture

```text
Le livrable vient apres une ouverture du pan philosophique, une extraction
epistemologique depuis les cycles physiques, puis une instruction minimale des
echelles philosophiques. La qualification donne un axe fort, et le potentiel du
corpus se mesure aussi par ses echelles d'acces, de realisme medie, de
validite, de reconstruction, de formation, de modalite, d'historicite, de
```
