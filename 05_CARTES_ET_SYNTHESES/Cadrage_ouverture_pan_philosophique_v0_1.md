## 1. Principe

Le pan philosophique du projet doit rester un espace d'exploration active.

Il a trois fonctions :

```text
1. extraire les problemes philosophiques deja contenus dans le travail physique ;
2. ouvrir des questions de second ordre sur la description, l'acces, la
   realite, la modalite, la temporalite et l'explication ;
3. situer le corpus dans plusieurs voisinages philosophiques, avec convergences,
   tensions fecondes et desaccords stricts.
```

Le pan philosophique sert donc deux mouvements :

```text
du corpus physique vers les questions philosophiques ;
des paysages philosophiques vers les reformulations possibles du corpus.
```

## 2. Theatre philosophique large

Le theatre philosophique du projet peut etre ouvert en dix foyers.

| Foyer | Question de second ordre | Rapport au corpus |
|---|---|---|
| Description ontologique prudente | Que decrit-on lorsque l'on qualifie une constante ? | Eviter l'essence forte et la derealisation |
| Epistemologie des acces | Comment une grandeur devient-elle connaissable, mesurable, fixable, bornee ou reconstruite ? | Regimes d'acces, routes, projections |
| Realisme medie | Comment parler de processus reels a travers des dispositifs d'acces situes ? | Ancrage realiste, Massimi, pluralite des acces |
| Coordination et mesure | Comment les grandeurs, unites, instruments et theories se coordonnent-ils ? | SI, Tal, Papale, van Fraassen, Reichenbach |
| Modalite | Qu'est-ce qui est necessaire, contingent, conventionnel, invariant ou stable ? | Dirac/Eddington, fine-tuning, constantes dimensionnelles |
| Temporalite et historicite | Comment une constance se stabilise-t-elle dans le temps ? | CODATA, regime gris, trajectoires, matrices temporelles |
| Constitution et emergence | Comment un regime rend-il certaines grandeurs disponibles ? | BEH, seuil, Saveur-Higgs, effectivite |
| Explication et limite | Que fait une limite : borne, validite, non-detection, tension, completion ? | Audit des regimes de limite, neutrinos, cosmologie |
| Architecture et holisme | Comment une grandeur tient-elle avec un reseau ? | Test de retrait, Duhem, Quine, Wimsatt |
| Institution et materialite | Comment les communautes, normes, instruments et realisations participent-ils a la stabilisation ? | SI, metrologie, CODATA, pratiques de mesure |

Cette table ouvre un champ et donne une orientation provisoire.

## 3. Rang des voisinages

Un voisinage philosophique gagne a etre rattache a une question precise, puis
explore dans son extension propre.

Exemple :

```text
Massimi sert deja l'axe realisme medie / pluralite des acces.
```

Mais Massimi peut aussi toucher :

```text
objectivite ;
robustesse ;
perspectivisme ;
modeles ;
conditions de representation ;
realite physique sous description.
```

Regle :

```text
un voisinage entre d'abord par une question,
puis il doit etre explore dans son extension propre.
```

Cela vaut aussi pour :

```text
Tal ;
Papale ;
van Fraassen ;
Reichenbach ;
Chang ;
Lange ;
Daston ;
Rheinberger ;
Wimsatt ;
Bitbol ;
Bachelard ;
Duhem ;
Quine ;
Whitehead ;
Peirce ;
Poincare.
```

## 4. Distinction entre fenetre et pan

La fenetre actuelle est :

```text
qualification des constantes effectives stabilisees ;
acces ;
realisme medie ;
Massimi.
```

Le pan philosophique est plus vaste :

```text
description du reel ;
conditions d'acces ;
statut de la mesure ;
modalite des constantes ;
temporalite de la stabilisation ;
formation des regimes ;
architecture des dependances ;
limites et non-detections ;
institutions metrologiques ;
conditions de presentation publique.
```

Regle :

```text
la fenetre active sert a travailler ;
le pan philosophique conserve l'espace des questions ouvertes.
```

## 5. Ordre de travail ouvert

L'ordre corrige devient :

```text
P0. cadrer le pan philosophique comme champ ouvert ;
P1. extraire les axes epistemologiques depuis les cycles physiques ;
P2. cartographier les foyers philosophiques larges ;
P3. rattacher les situations historiques comme resonances ;
P4. explorer les voisinages au-dela de leur question d'entree ;
P5. instruire les axes structurants et les axes de profondeur ;
P6. comparer les voies de livrable ;
P7. choisir un premier texte.
```

La sequence garde une discipline :

```text
ouvrir le champ ;
extraire depuis le corpus ;
chercher les voisinages ;
tester les formulations ;
choisir ensuite.
```

## 6. Effet sur l'extraction epistemologique

L'extraction epistemologique depuis les cycles physiques garde sa valeur.

Nouveau rang :

```text
entree interne par le materiau physique.
```

Elle maintient le pan philosophique ouvert.

Elle fournit :

```text
un noyau de questions deja soutenues par le corpus ;
une protection contre les formulations trop generales ;
un point de depart pour les voisinages.
```
