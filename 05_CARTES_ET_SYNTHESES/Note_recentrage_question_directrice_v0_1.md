# Note de recentrage sur la question directrice v0.1

## Constantes d'abord, methode ensuite

### 1. Fonction

Cette note corrige un risque apparu dans la matrice des acquis theoriques :

```text
l'extraction fonctionne,
mais elle peut faire remonter trop d'elements de methode interne
au meme niveau que les acquis portant sur les constantes.
```

La question directrice doit rester :

```text
dans quelle mesure une grandeur peut-elle etre qualifiee de constante ?
```

### 2. Distinction de rang

Il faut distinguer trois rangs.

| Rang | Fonction | Place dans la future these |
|---|---|---|
| Question directrice | Qualifier le statut de constance d'une grandeur | Centre de l'argument |
| Methode externe | Expliquer comment cette qualification est etablie | Appui argumentatif |
| Methode interne | Organiser le corpus, les fichiers et les procedures | Outillage, non contenu principal |

La methode interne est indispensable au travail.

Mais elle ne doit pas devenir le sujet de la these.

### 3. Test de passage vers la these

Avant d'integrer un acquis au premier degagement de these, poser trois
questions.

```text
1. Cet acquis dit-il quelque chose du statut d'une constante ?
2. Aide-t-il a comprendre pourquoi une grandeur tient comme constante ?
3. Peut-il etre formule sans parler d'abord de notre procedure interne ?
```

Si la reponse est non, l'acquis doit rester dans l'appareil methodologique.

S'il repond oui a au moins une des deux premieres questions, il peut entrer
dans le degagement de these, avec prudence.

### 4. Correction appliquee a la matrice

La matrice des acquis doit etre lue ainsi :

```text
acquis centraux sur les constantes :
A1, A3, A4, A5, A12, A13

cas physiques porteurs :
A8, A9, A10, A11

articulation architecturale :
A6

outillage methodologique :
A2, A7, A14
```

Les acquis methodologiques ne sont pas elimines.

Ils sont remis a leur rang.

### 5. Formule de controle

Formule a conserver pour la suite :

```text
La methode sert a etablir dans quelle mesure une grandeur peut etre dite constante ;
elle ne doit pas remplacer cette question.
```

### 6. Suite

Le premier degagement de these devra donc commencer par les constantes, non par
la methode.

Il pourra ensuite montrer pourquoi la methode etait necessaire pour rendre la
qualification rigoureuse.
