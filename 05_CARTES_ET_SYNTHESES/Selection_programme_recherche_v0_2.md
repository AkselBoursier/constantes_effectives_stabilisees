# Sélection du programme de recherche v0.2

## Statut

```text
succède pour l’usage courant : Selection_programme_recherche_v0_1.md ;
standard : contribution à des conversations existantes ;
contrôle secondaire : nouveauté seulement lorsqu’elle est revendiquée ;
condition : aucune thèse générale avant positionnement et tests.
```

## Critères communs

Chaque chantier doit établir :

```text
C1 — conversation réelle ;
C2 — recouvrement honnête ;
C3 — différenciation défendable ;
C4 — limites conservées.
```

Les niveaux deviennent :

```text
Q0 — correction et documentation ;
Q1 — coordination de couches ;
Q2 — contribution différenciée ;
Q3 — protocole transférable ;
Q4 — nouveauté substantielle éventuelle, non requise.
```

## Chantier Q — Catégories, accès et inférences en microphysique

Question :

> Quelles fonctions les catégories exercent-elles dans la préparation, le contrôle, l’enregistrement, l’inférence et la communication, et quelles inférences deviennent illégitimes lorsqu’une catégorie migre entre ces fonctions ?

Conversations engagées : concepts classiques, théorie des instruments, complémentarité, philosophie de l’expérience, représentation et pédagogie quantique.

Contribution visée : une matrice croisant :

```text
catégorie
x fonction
x type d’accès
x inférence autorisée ou interdite.
```

Tests produits : Stern–Gerlach ; séquences d’axes ; interférence et marquage des voies.

Statut : `Q2 plausible`, à positionner bibliographiquement. L’absence de nouveauté physique n’invalide pas cette contribution clarificatrice.

## Chantier M — Portée explicative des mécanismes

Question :

> Lorsqu’un mécanisme est présenté comme une explication, explique-t-il une valeur, une relation, un domaine d’opérativité, une robustesse ou le transfert d’une dépendance ?

Conversations engagées : mécanismes, explication, EFT, naturalité, brisure de symétrie et dépendances de modèles.

Contribution visée : matrice intersectorielle comparant objet expliqué, relation dérivée, valeur libre, domaine, corrections, contingence déplacée et dette nouvelle.

Premier test prévu : `G_F`.

## Chantier A — Contexte, agents et preuve

Question :

> Comment le choix, le rang et l’ordre des documents modifient-ils le problème identifié par un agent, les sources mobilisées, la modalité des conclusions et la révision d’erreur ?

Conversations engagées : ingénierie du contexte, provenance, cognition distribuée, épistémologie sociale et scientific software.

Contribution visée : protocole comparatif reproductible sur des paquets de contexte contrôlés.

## Ordre révisé

```text
1. positionner bibliographiquement la contribution Q1 ;
2. appliquer M1 à G_F ;
3. concevoir A1 sans lancement massif ;
4. comparer la forme des contributions Q, M et A ;
5. choisir le programme principal selon utilité, robustesse et faisabilité.
```

## Règle de décision

Une question n’est pas abandonnée parce qu’elle est occupée. Elle est abandonnée ou déplacée si :

```text
la conversation n’est pas identifiable ;
le travail ne dépasse pas une redite involontaire ;
la différenciation ne peut pas être formulée ;
les limites rendent la contribution vide ou trompeuse ;
le coût excède manifestement l’utilité attendue.
```

La nouveauté demeure une possibilité à constater, jamais une obligation à produire.
