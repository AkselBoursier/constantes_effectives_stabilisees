## 2. Quatre niveaux a maintenir separes

La separation active du corpus peut etre formulee en quatre niveaux.

| Niveau | Question | Fonction |
|---|---|---|
| Regime d'effectuation | Ou la grandeur, l'operation ou la structure opere-t-elle ? | Ancrage physique, effectif, definitionnel ou operationnel |
| Regime d'acces | Par quelles routes devient-elle connue, mesuree, fixee, reconstruite ou bornee ? | Mediation epistemique explicite |
| Regime de qualification | Dans quelle mesure son statut de constance est-il stabilise ? | Travail propre du corpus |
| Positionnement philosophique | Quelle description de second ordre rend cette operation intelligible ? | Dialogue avec epistemologie et philosophie de la physique |

Formule de travail :

```text
decrire le regime d'effectuation ;
nommer les acces ;
qualifier la constance ;
puis seulement positionner philosophiquement.
```

Cette sequence donne une protection contre deux lectures symetriques :

```text
substantialisation des constantes comme essences independantes de tout acces ;
lecture derealisante ou l'acces absorberait l'objet.
```

## 4. Axes epistemologiques extraits

### 4.1. Ancrage realiste medie

Les cycles decrivent des grandeurs, operations et structures qui renvoient a
des regimes reels de la physique ou de la mesure.

Apport :

```text
le corpus parle du reel sous conditions de regime et d'acces.
```

Point philosophique :

```text
description realiste mediee des statuts de constance.
```

Voisinage a instruire :

```text
Massimi : realisme perspectival, robustesse situee, pluralite des acces.
```

Question de friction :

```text
le corpus pourrait distinguer regime physique et regime d'acces plus fortement
que certaines lectures perspectivistes.
```

### 4.2. Acces comme qualification du statut

Tous les cycles confirment que l'acces est indispensable.

Mais l'acces change de poids selon les cas :

```text
central dans le SI ;
central mais modele-indexe en cosmologie ;
qualifiant dans Saveur-Higgs ;
porteur de validite dans l'effectif ;
heterogene et partiel dans les neutrinos.
```

Apport :

```text
l'acces qualifie le statut de constance d'une grandeur ancree.
```

Question de second ordre :

```text
comment formuler une epistemologie des acces qui garde l'ancrage du processus
decrit ?
```

### 4.3. Validite de description

Le cycle effectif basse energie donne l'axe le plus net :

```text
une description limitee peut etre positive, robuste et pertinente.
```

Apport :

```text
validite de domaine ;
non-resolution explicite ;
description operatoire ;
limite conservee.
```

Question philosophique :

```text
comment penser une realite de description avec la completion conservee comme
horizon ?
```

### 4.4. Reconstruction et inference

La cosmologie donne l'axe le plus fort de reconstruction :

```text
une grandeur peut etre fortement stabilisee dans un modele,
par plusieurs acces,
avec une interpretation physique encore ouverte.
```

Apport :

```text
modele d'inference ;
observable reconstruite ;
tension d'acces ;
sous-reseau fonctionnel.
```

Question philosophique :

```text
quel type de realisme convient aux grandeurs reconstruites et modele-indexees ?
```

### 4.5. Definition et realisation

Le SI donne l'axe local de definition :

```text
une valeur peut etre exacte par definition,
avec une realisation pratique encore incertaine,
et un role physique conserve.
```

Apport :

```text
valeur definissante ;
composition exacte ;
realisation ;
role physique.
```

Question philosophique :

```text
comment une convention devient-elle une operation de stabilisation plutot
qu'un arbitraire ?
```

### 4.6. Borne, projection et acces heterogenes

Les neutrinos donnent l'axe le plus sensible :

```text
un objet physique peut etre robuste par certains traits,
borne par d'autres,
et projete differemment selon les acces.
```

Apport :

```text
ecarts spectraux ;
ancrage absolu borne ;
projection cinematique ;
projection cosmologique ;
projection Majorana conditionnelle.
```

Question philosophique :

```text
comment qualifier un statut de constance quand l'objet est connu par
projections non equivalentes ?
```

### 4.7. Constitution et seuil

Saveur-Higgs et BEH donnent l'axe de constitution :

```text
une grandeur peut rendre un regime disponible,
avec l'origine profonde de ses parametres conservee comme horizon.
```

Apport :

```text
seuil ;
constitution ;
disponibilite de regime ;
solidarite des masses, couplages et orientations.
```

Question philosophique :

```text
comment decrire une constitution locale avec son horizon d'explication
conserve ?
```

### 4.8. Architecture comme solidarite testable

Les architectures transversales donnent l'axe de solidarite :

```text
une constante tient souvent avec d'autres grandeurs, fonctions et regimes.
```

Apport :

```text
test de retrait ;
dependances actives ;
solidarite locale ;
degres de dissolution ou de degradation.
```

Question philosophique :

```text
comment faire du holisme un objet de mesure plutot qu'un principe general ?
```

## 6. Massimi comme point de verification local

Massimi devient un voisinage a instruire localement pour l'axe suivant :

```text
realisme medie, pluralite des acces et robustesse situee.
```

Points de convergence probables :

```text
pluralite des acces ;
robustesse situee ;
description situee ;
realite physique conservee.
```

Points de tension possibles :

```text
le corpus distingue explicitement regime d'effectuation et regime d'acces ;
le corpus travaille le statut de constance d'une grandeur plutot qu'une theorie
generale de la perspective ;
le corpus donne un role fort aux limites, bornes, realisations et architectures.
```

Decision :

```text
Massimi est un voisinage local de verification,
avec convergence, tension feconde et desaccord strict a distinguer.
```

Le pan philosophique garde d'autres voisinages a ouvrir selon les foyers :

```text
coordination et mesure ;
modalite ;
historicite ;
constitution ;
architecture ;
institution metrologique ;
explication et limites.
```
