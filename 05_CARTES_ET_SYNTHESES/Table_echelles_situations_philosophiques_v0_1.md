## 2. Regle de lecture

Une situation peut nourrir plusieurs echelles.

La table distingue :

```text
echelle primaire : ce que la situation instruit directement ;
echelles secondaires : ce qu'elle nourrit par raccord ;
apport : ce que la situation ajoute au corpus ;
maturite : statut P1-P6 du resultat utilisable ;
effet livrable : A1, A2, A4, B ou horizon.
```

Statuts de maturite :

```text
P1 : acquis mobilisable ;
P2 : formule stable ;
P3 : instrument utile ;
P4 : hypothese candidate ;
P5 : programme ouvert ;
P6 : horizon de recherche.
```

## 8. Diagnostic provisoire

La table confirme trois points :

```text
1. la qualification est l'axe le plus lisible et forme une entree parmi
   plusieurs echelles philosophiques ;
2. la metrologie et l'historicite forment le bloc le plus dense ;
3. formation genetique, modalite et constitution portent l'originalite
   longue, avec une maturite plus basse.
```

Hypothese provisoire :

```text
A1 est le candidat tactique le plus solide ;
A2 reste le candidat theorique le plus naturel ;
B porte probablement l'originalite philosophique la plus forte,
mais demande un cycle d'instruction propre ;
A4 sert de test de frontiere et de livrable methodologique possible.
```
