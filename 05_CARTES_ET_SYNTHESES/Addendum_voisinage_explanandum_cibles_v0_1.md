# Addendum — explananda, contrastes et déplacements de cible v0.1

## 0. Statut

```text
statut : correction du reste candidat après recherche complémentaire ;
date : 21 juillet 2026 ;
fonction : tester l’idée selon laquelle le projet détecterait des questions adressées au
           mauvais objet ;
ne vaut pas : revue exhaustive de la philosophie de l’explication.
```

## 1. Reste candidat avant contrôle

La première passe avait isolé la possibilité suivante :

> La comparaison des constantes entre secteurs pourrait révéler que certaines demandes
> d’explication visent le mauvais objet : une valeur plutôt qu’une relation, une propriété
> plutôt qu’une convention, ou une origine plutôt qu’un domaine de validité.

Cette formulation semblait plus spécifique que les thèses générales sur la pluralité des
modèles, la mesure ou l’invariance.

## 2. Voisinage direct

### 2.1 Explication contrastive

La tradition de l’explication contrastive soutient qu’une question explicative doit être
comprise comme :

```text
pourquoi P plutôt que Q ?
```

Le choix du contraste détermine ce qui compte comme pertinent pour l’explication. Une
question mal contrastée peut donc appeler une réponse correcte mais inadéquate à l’intérêt
réel de l’enquête.

### 2.2 Conceptualisation de l’explanandum

Les conceptions représentationnelles ou « eikoniques » de l’explication insistent sur le
fait que l’on n’explique pas un phénomène entièrement indépendant de sa conceptualisation.
L’explanandum est formulé dans un programme de recherche et sous des contraintes
épistémiques particulières.

### 2.3 Déplacement non reconnu de cible

Une littérature récente distingue le changement légitime d’explanandum d’un problème plus
précis : fournir une explication d’une cible déplacée tout en croyant encore expliquer la
cible initiale. L’évaluation de l’explanans seul ne suffit pas ; il faut comparer les
conséquences empiriques et les contrastes associés aux deux explananda.

## 3. Recouvrement avec le corpus

Le projet rencontre exactement cette structure lorsqu’il distingue :

```text
expliquer la valeur d’un coefficient ;
expliquer la forme d’une relation ;
expliquer la validité d’une approximation ;
expliquer la précision d’une estimation ;
expliquer la fixation conventionnelle d’une valeur numérique ;
expliquer l’histoire ou le mécanisme d’un objet physique.
```

Dire qu’une question sur « l’origine d’une constante » doit parfois être reformulée ne
constitue donc pas, à lui seul, une contribution originale.

## 4. Reste encore possible

Le reste doit être réduit à une proposition opérationnelle :

> Une matrice spécialisée sur les objets appelés constantes permet-elle de détecter de
> manière reproductible un déplacement non reconnu de l’explanandum, puis de montrer que
> ce déplacement modifie une expérience, un calcul, une interprétation ou une décision de
> recherche ?

L’originalité éventuelle porterait sur :

```text
la sélection du domaine des constantes ;
la procédure de diagnostic ;
la reproductibilité des verdicts ;
les conséquences produites dans plusieurs secteurs.
```

Elle ne porterait pas sur l’idée générale de cible explicative ou de contraste.

## 5. Test renforcé

Pour chaque cas candidat, le dossier devra contenir :

```text
E_original : phénomène ou objet initialement demandé ;
Q_original : contraste implicite ou explicite ;
X_original : explication recherchée ;
E_shifted : objet effectivement expliqué par l’analyse ;
Q_shifted : contraste associé ;
conséquences empiriques distinctes ;
preuve que le déplacement n’était pas déjà reconnu ;
effet de la reconnaissance sur la recherche.
```

Sans différence de conséquences ou de contraste, il n’y a pas de déplacement substantiel.

## 6. Application provisoire aux cas

### QCD

```text
E_original possible : pourquoi alpha_s possède-t-il cette valeur ?
E_shifted dans le corpus : comment plusieurs extractions sont-elles rapportées à m_Z ?
```

Le déplacement est réel, mais il n’est pas démontré que la littérature QCD confonde ces
questions. Le corpus a peut-être simplement choisi une question différente.

### G_F

```text
E_original possible : pourquoi G_F a-t-il cette valeur ?
E_shifted : pourquoi une interaction locale décrit-elle la basse énergie ?
```

La théorie électrofaible et l’EFT distinguent déjà clairement ces explananda. Aucun
problème non reconnu n’est établi.

### SI

```text
E_original possible : pourquoi la constante a-t-elle une valeur exacte ?
E_shifted : pourquoi cette valeur numérique définit-elle une unité et comment est-elle
            réalisée ?
```

La littérature métrologique reconnaît explicitement ce déplacement. Le projet ne le
découvre pas.

### Cycle 10

```text
E_original : comment une variation ancienne peut-elle devenir faible ?
E_shifted après contrôle : comment l’énergie du champ est-elle éliminée ou transférée ?
```

Ici, le déplacement a une conséquence physique et a modifié le programme de calcul. Mais
il provient du bilan énergétique du modèle. Il faut encore montrer ce que l’étude des
constantes ajoute à cette requalification.

## 7. Verdict

```text
idée générale du déplacement de cible : occupée ;
application aux constantes : thématique, non originale par elle-même ;
instrumentation comparative reproductible : non démontrée ;
cas le plus prometteur : cycle 10, mais gain transversal non établi ;
conséquence : le dernier reste théorique est davantage réduit.
```
