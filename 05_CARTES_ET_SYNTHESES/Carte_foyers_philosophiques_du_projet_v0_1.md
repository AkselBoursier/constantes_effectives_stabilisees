## 1. Fonction de la carte

Le pan philosophique du projet s'ouvre par plusieurs entrees.

Il articule les foyers suivants :

```text
description du reel ;
acces ;
mesure ;
modalite ;
temporalite ;
constitution ;
limite ;
architecture ;
institution ;
presentation publique.
```

La carte sert a :

```text
1. rendre ces foyers visibles ;
2. indiquer ce que le corpus a deja produit pour chacun ;
3. signaler les voisinages philosophiques pertinents ;
4. separer acquis, pistes, dettes de lecture et horizons ;
5. preparer les prochaines fiches d'instruction.
```

## 3. Lecture par familles de questions

### 3.1. Foyers descriptifs

Ces foyers disent ce que le corpus decrit.

```text
description ontologique prudente ;
realisme medie ;
constitution et emergence ;
formation genetique.
```

Fonction :

```text
tenir ensemble ancrage realiste, mediation par acces, constitution locale
et horizons d'explication.
```

Question centrale :

```text
comment parler de processus reels en tenant les constantes comme statuts,
regimes et trajectoires qualifies ?
```

### 3.2. Foyers epistemologiques

Ces foyers disent comment le corpus connait, mesure, reconstruit ou borne.

```text
epistemologie des acces ;
coordination et mesure ;
reconstruction et inference ;
explication et limite.
```

Fonction :

```text
decrire les routes par lesquelles une grandeur devient qualifiable comme
constante.
```

Question centrale :

```text
comment l'acces qualifie-t-il un statut tout en maintenant l'objet qualifie
comme ancrage ?
```

### 3.3. Foyers temporels et modaux

Ces foyers disent comment une constance se stabilise et quel type de necessite
elle engage.

```text
temporalite ;
historicite ;
modalite ;
latitude ;
convention ;
invariance.
```

Fonction :

```text
suivre les trajectoires de stabilisation et les degres de necessite.
```

Question centrale :

```text
quand une constante devient-elle stable, exacte, conventionnable, bornee,
ou seulement robuste dans un regime ?
```

### 3.4. Foyers architecturaux et institutionnels

Ces foyers disent comment les grandeurs tiennent ensemble et comment leurs
conditions pratiques se stabilisent.

```text
architecture ;
holisme ;
institution ;
materialite ;
realisations ;
standards.
```

Fonction :

```text
decrire les reseaux de dependance, les tests de retrait et les milieux
institutionnels de stabilisation.
```

Question centrale :

```text
qu'est-ce qui tient quand une constante tient : une valeur, un acces, une
architecture, une pratique, un regime ou leur solidarite ?
```

### 3.5. Foyers de frontiere

Ces foyers testent l'amplitude du cadre.

```text
exportabilite ;
frontiere du domaine ;
presentation publique ;
livrables.
```

Fonction :

```text
mesurer ce que la grille permet de comparer, et ce qu'elle doit reserver.
```

Question centrale :

```text
comment presenter une grammaire de qualification comme cadre borne et ouvert ?
```
