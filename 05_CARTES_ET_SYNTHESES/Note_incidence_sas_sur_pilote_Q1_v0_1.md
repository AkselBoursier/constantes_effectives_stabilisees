# Note d'incidence du sas critique sur le pilote Q1 v0.1

## 0. Statut

```text
statut : préparation du pilote, sans lancement de l'analyse Stern–Gerlach ;
appuis : rectification de l'accès, registre des cinq lectures, réserve pédagogique ;
fonction : empêcher que Q1 hérite des contresens détectés.
```

## 1. Question Q1 maintenue

> Quelles fonctions distinctes les catégories classiques exercent-elles dans la
> préparation, le contrôle, l'enregistrement, l'inférence et la communication d'une
> expérience quantique, et quand deviennent-elles trompeuses comme attributions
> ontologiques ?

Le pilote ne cherchera pas à prouver une théorie générale de la mesure, de la
coupure ou de la constitution des propriétés.

## 2. Accès à analyser dans Stern–Gerlach

```text
accès matériel : source, collimation, aimant, gradient, écran ou détecteur ;
accès opératoire : orientation, préparation et répétition ;
accès représentationnel : axe, état, opérateur, probabilités ;
accès inférentiel : passage des impacts aux affirmations sur le spin ;
accès intersubjectif : description reproductible du dispositif et des résultats ;
accès pédagogique : images et catégories utilisées pour introduire l'expérience.
```

Ces dimensions ne doivent pas être fusionnées sous `appareil` ou `contexte`.

## 3. Catégories candidates

```text
particule ;
trajectoire ;
faisceau ;
orientation ;
axe ;
position ;
impact ;
résultat ;
état ;
propriété ;
mesure ;
séparation en deux classes.
```

Pour chacune, le pilote demandera :

```text
fonction préparatoire ;
fonction opératoire ;
fonction d'enregistrement ;
fonction inférentielle ;
fonction communicationnelle ;
fonction pédagogique ;
engagement ontologique éventuel ;
point où la catégorie devient trompeuse.
```

## 4. Coupes à distinguer

```text
partition matérielle du dispositif ;
séparation système / appareil ;
partition formelle dans le modèle ;
choix d'un axe de mesure ;
coarse-graining des impacts ;
découpage narratif ou pédagogique ;
coupe documentaire entre preuve et interprétation.
```

Le terme `coupe` ne sera pas utilisé avant que son type soit spécifié.

## 5. Garde-fous pédagogiques

```text
aucune toupie pour représenter le spin ;
aucune petite sphère en rotation ;
aucune trajectoire classique présupposée entre source et impact ;
aucune conversion immédiate de deux taches en propriété classique préexistante ;
aucune assimilation de l'expérience à une stabilisation de constante.
```

La réserve pédagogique pourra être utilisée pour étudier le retrait progressif
des images classiques, non pour expliquer le spin par analogie.

## 6. Chaîne de transition attendue

```text
dispositif et protocole
-> distribution d'impacts
-> représentation formelle
-> inférences autorisées
-> attribution éventuelle au système
-> interprétations concurrentes
-> portée philosophique bornée.
```

À chaque passage seront indiquées les opérations supplémentaires requises.

## 7. Résultat discriminant attendu

Q1 sera intéressant si la matrice montre qu'une catégorie peut être :

```text
indispensable dans la préparation ;
remplaçable ou transformée dans le calcul ;
nécessaire à l'enregistrement ;
utile à la communication ;
dangereuse dans l'interprétation ontologique.
```

Échec du pilote : toutes les catégories reçoivent le même profil, ou la matrice
ne fait que répéter que « le langage classique est nécessaire ».

## 8. Dettes avant lancement

```text
recherche d'antériorité approfondie sur Bohr, concepts classiques et Stern–Gerlach ;
description physique précise du dispositif réel et de ses idéalisations ;
distinction spin, moment magnétique et déflexion spatiale ;
choix explicite du niveau : expérience historique, schéma pédagogique ou dispositif moderne ;
identification des interprétations comparées ;
critères de non-trivialité de la matrice.
```

## 9. Incidence documentaire

Le pilote devra comporter deux couches :

```text
scène principale : expérience, catégories, fonctions et résultats ;
provenance : sources, décisions, matrices, objections et audit.
```

> Le sas ne fournit pas encore la réponse à Q1. Il précise ce que le pilote doit
> distinguer pour ne pas transformer Stern–Gerlach en illustration d'une doctrine
> préexistante.
