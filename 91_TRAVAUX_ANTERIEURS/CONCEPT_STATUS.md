# CONCEPT_STATUS.md

Statut : registre de phase 1.

Ce fichier synthétise le statut actuel des principaux concepts. Il ne remplace pas `DECISION_LEDGER.md`.

## Légende

- `provisoire` : conservé comme outil de travail, non stabilisé définitivement ;
- `validé restreint` : autorisé sous conditions explicites ;
- `régionalisé` : autorisé dans certains domaines seulement ;
- `suspendu` : ne pas employer comme prémisse ;
- `retiré` : ne pas réintroduire sauf mention historique ou critique ;
- `critère` : règle méthodologique, documentaire ou évaluative ;
- `hypothèse régionale` : proposition instruisable dans un domaine déterminé ;
- `terrain d'épreuve` : cas utilisé pour tester un concept ou un seuil ;
- `différé` ou `à reconstruire` : fonction conservée, architecture ou dossier à traiter plus tard.
