## 1. Principe général d'architecture

La v13 ne doit plus être construite comme une théorie générale des coupes constitutives.

Elle doit fonctionner comme une architecture d'enquête comparative :

1. établir les conditions de l'enquête ;
2. distinguer les termes de base ;
3. remplacer la cascade v12 par une matrice multidimensionnelle ;
4. analyser séparément les relations, les temporalités, les échelles et les devenirs épistémiques ;
5. tester le gain non redondant du vocabulaire du cadre ;
6. conserver la possibilité d'une fragmentation ou d'un échec local.

Le document final ne doit pas raconter une progression du physique vers le réflexif. Il doit fournir des instruments de comparaison capables de respecter les mécanismes locaux.

## 3. Ordre de construction

L'ordre logique du document final n'est pas identique à l'ordre de travail.

Ordre de travail recommandé :

1. phase 3 : créer K comme registre provisoire de contrôle ;
2. phase 4 : réécrire A à J une section à la fois ;
3. phase 5 : reconstruire L et l'annexe ;
4. phase 6 : finaliser K à partir des sections réécrites et des voisinages ;
5. phase 7 : reconstruire M ;
6. phase 8 : audit transversal ;
7. phase 9 : synchronisation du manuscrit.

K doit donc exister tôt comme instrument de contrôle, mais ne doit pas être stabilisé avant L et l'annexe. M vient ensuite comme registre de migration à partir de K stabilisé.

## 4. Architecture détaillée des sections A à J

### A. Fonction du cadre et contrat d'enquête

Fonction :

- présenter la v13 comme programme comparatif ;
- poser la question directrice : que fait une différenciation déterminée à ce qu'elle distingue, selon quels mécanismes, avec quelles asymétries et quelle persistance ;
- définir la discipline modale du cadre ;
- rendre explicite la possibilité de retrait, fragmentation ou échec.

Dépendances :

- D-001, D-002, D-008, D-013, D-014, D-022.

Concepts autorisés :

- programme comparatif ;
- différenciation déterminée ;
- effet variable ;
- hypothèse du cadre ;
- reconstruction disciplinaire ;
- gain non redondant ;
- échec local.

Concepts interdits ou suspendus :

- théorie générale des coupes constitutives ;
- toute coupe est constitutive ;
- cadre transversal déjà acquis ;
- voie médiane programmée ;
- totalisation métaphysique.

Études de cas requises :

- aucune étude de cas développée dans A ;
- exemples seulement indicatifs, sans fonction démonstrative.

### B. Vocabulaire de base

Fonction :

- établir les distinctions minimales ;
- fixer le statut provisoire de "coupe" ;
- distinguer effet de sélection, opérationnalisation, transformation, stabilisation, individuation et constitution.

Dépendances :

- A ;
- D-002, D-003, D-004, D-005.

Concepts autorisés :

- différenciation ;
- distinction ;
- frontière ;
- interaction ;
- couplage ;
- constitution comme effet fort ;
- coupe comme terme de travail.

Concepts interdits ou suspendus :

- synonymie entre distinction et constitution ;
- coupe comme invariant démontré ;
- constitution universelle ;
- production intégrale des termes par défaut.

Études de cas requises :

- au moins un cas de différenciation sans observateur ;
- au moins un cas de distinction expérimentale ou méthodologique ;
- au moins un cas institutionnel ou classificatoire.

### C. Matrice multidimensionnelle d'analyse

Fonction :

- remplacer la cascade v12 par une matrice d'axes indépendants ;
- traiter les cas selon domaine, opération, explicitation, effet, temporalité, dépendance et capacité de retour.

Dépendances :

- A et B ;
- D-007, D-009, D-010, D-012, D-015.

Concepts autorisés :

- domaine ou site ;
- mode d'opération ;
- statut d'explicitation ;
- effet principal ;
- temporalité ;
- structure de dépendance ;
- capacité de retour sur l'opération.

Concepts interdits ou suspendus :

- gradient physique-biologique-réflexif ;
- pré-thématique comme type stable de coupe ;
- réflexivité comme sommet d'une cascade ;
- orthogonalité proclamée sans test.

Études de cas requises :

- un cas physique ou physicochimique ;
- un cas expérimental ou métrologique ;
- un cas cognitif, social ou institutionnel.

### D. Relations et asymétries

Fonction :

- déplier les relations sans les ramener à la co-constitution ;
- distinguer interaction, couplage, co-détermination, co-stabilisation, co-individuation et co-constitution ;
- analyser les asymétries.

Dépendances :

- B et C ;
- D-005, D-006.

Concepts autorisés :

- interaction ;
- couplage qualifié ;
- co-détermination ;
- co-stabilisation ;
- co-individuation ;
- co-constitution comme hypothèse maximale ;
- asymétries causale, matérielle, temporelle, de dépendance, de contrôle, de vulnérabilité, épistémique et institutionnelle.

Concepts interdits ou suspendus :

- co-constitution comme nom général du lien ;
- réciprocité comme symétrie ;
- couplage structurel comme preuve transversale ;
- relation sans mécanisme spécifié.

Études de cas requises :

- couplage structurel biologique reconstruit dans son terrain propre ;
- cas de relation réciproque mais asymétrique ;
- cas où la co-constitution doit être refusée malgré une interaction.

### E. Temporalités, formation, maintien et transformation

Fonction :

- remplacer les "destins" v12 par une analyse des temporalités ;
- distinguer apparition, stabilisation, maintien, transformation, latence, disparition, héritage et réapparition.

Dépendances :

- B, C, D ;
- D-015, D-016, D-017, D-018, D-019.

Concepts autorisés :

- apparition ;
- stabilisation ;
- maintien ;
- transformation ;
- latence ;
- disparition ou rupture ;
- héritage ;
- sédimentation régionalisée.

Concepts interdits ou suspendus :

- destins universels ;
- solidification transversale ;
- porosité comme phase commune ;
- dé-sédimentation comme envers général ;
- dormance hors biologie sans justification.

Études de cas requises :

- membrane biologique ou autre frontière maintenue ;
- cas historique ou institutionnel de sédimentation régionale ;
- cas où persistance n'implique pas sédimentation.

### F. Échelles, résolution, stabilité et persistance

Fonction :

- préciser comment une structure apparaît selon les profils de résolution ;
- distinguer accès descriptif, stabilité réelle et mécanisme de maintien ;
- pluraliser les formes de stabilité.

Dépendances :

- C et E ;
- D-015, D-016, D-019.

Concepts autorisés :

- résolution temporelle, spatiale, énergétique, instrumentale, statistique et théorique ;
- coarse-graining ;
- stabilité apparente ;
- stabilité dynamique ;
- équilibre ;
- métastabilité ;
- état stationnaire hors équilibre ;
- robustesse ;
- maintenance active ;
- reproduction.

Concepts interdits ou suspendus :

- stabilité produite par le regard seul ;
- triade terminer-tenir-suspendre ;
- axe unique des modes de persistance ;
- confusion entre échelle d'observation et mécanisme.

Études de cas requises :

- cas physique où la résolution change ce qui devient visible ;
- cas biologique ou technique de maintien actif ;
- cas de stabilité apparente distincte d'une stabilité dynamique.

### G. Devenirs épistémiques, recouvrement et réflexivité

Fonction :

- analyser les devenirs des opérations de connaissance ;
- distinguer tacitation, boîte noire, oubli, naturalisation, objectivation, réification et recouvrement ;
- réserver la réflexivité critique aux pratiques capables d'explicitation et de révision.

Dépendances :

- C, E, F ;
- D-009, D-010, D-011, D-020, D-021.

Concepts autorisés :

- tacitation ;
- boîte noire ;
- oubli historique ;
- naturalisation ;
- objectivation ;
- réification ;
- recouvrement cognitif restreint ;
- réflexivité critique ;
- révisabilité méthodologique.

Concepts interdits ou suspendus :

- recouvrement comme toute opacité ;
- objectivation comme faute automatique ;
- chaîne sédimentation vers recouvrement vers réification comme destin universel ;
- gradient des rapports à soi ;
- dichotomie vivant sanctionné contre enquête non sanctionnée.

Études de cas requises :

- cas de médiation encore opérante devenue invisible ;
- cas de boîte noire légitime ;
- cas d'objectivation robuste sans réification.

### H. Méthode d'application, gain non redondant et échec

Fonction :

- formaliser le passage du régime reconstructif au régime réflexif ;
- intégrer les statuts modaux ;
- définir les résultats possibles de l'application du cadre ;
- fixer le test de non-redondance et les critères d'échec.

Dépendances :

- A à G ;
- D-008, D-012, D-013, D-014, D-022, D-024.

Concepts autorisés :

- reconstruction indépendante ;
- hypothèse du cadre ;
- analogie de travail ;
- résultat scientifique robuste ;
- résultat dépendant d'un modèle ;
- débat disciplinaire ;
- proposition philosophique existante ;
- extension métaphysique ;
- redescription, synthèse, diagnostic, générativité, explication ou prédiction ;
- retrait, suspension, fragmentation, inapplicabilité, déformation, réfutation locale.

Concepts interdits ou suspendus :

- traduction immédiate d'un terrain dans le vocabulaire du cadre ;
- variation locale comme preuve ;
- souplesse méthodologique comme protection contre l'échec ;
- analogie comme conclusion.

Études de cas requises :

- la constante de structure fine alpha comme cas régional prioritaire possible ;
- un cas négatif où le cadre n'apporte aucun gain ;
- un cas où l'usage du vocabulaire du cadre déformerait le terrain.

### I. Héritage, contraintes cumulatives et réouvertures

Fonction :

- traiter ce qui demeure opérant d'un passé sans supposer une sédimentation universelle ;
- distinguer héritage, trace, transmission, contrainte cumulative, disponibilité et réactivation ;
- étudier les conditions où un passé modifie les opérations ultérieures.

Dépendances :

- E, F, G, H ;
- D-017, D-022, D-024.

Concepts autorisés :

- héritage ;
- contrainte cumulative ;
- support de conservation ;
- transmission ;
- disponibilité ;
- réactivation ;
- réouverture critique ;
- dépendance historique à préciser.

Concepts interdits ou suspendus :

- sédimentation cumulative universelle ;
- récit cosmique des coupes ;
- rétro-lisibilité comme preuve de nécessité ;
- trace assimilée à mémoire ;
- passé opérant sans support ni mécanisme.

Études de cas requises :

- cas historique ou épistémique de transmission ;
- cas où une trace ne suffit pas à produire un héritage ;
- cas de réactivation ou réouverture critique.

### J. Concepts régionaux et zones d'incubation

Fonction :

- accueillir les notions utiles mais non transversales ;
- empêcher leur promotion silencieuse au noyau général ;
- organiser les conditions de test, extension, retrait ou fragmentation.

Dépendances :

- A à I ;
- D-006, D-012, D-017, D-023, D-024, D-029.

Concepts autorisés :

- concept régional ;
- terrain d'épreuve ;
- invariant minimal ;
- règle de traduction ;
- seuil de rupture ;
- cas négatif ;
- autopoïèse comme notion biologique régionale ;
- couplage structurel comme notion biologique régionale ;
- dormance comme famille biologique régionale ;
- légalité effective comme hypothèse régionale.

Concepts interdits ou suspendus :

- concept régional traité comme preuve du noyau ;
- migration par ressemblance lexicale ;
- réflexivité conditionnelle comme thèse générale ;
- quantum comme preuve définitionnelle ;
- légifération universelle.

Études de cas requises :

- au moins un concept vivant ou biologique régional ;
- au moins un concept épistémique ou institutionnel régional ;
- au moins un concept physique ou formel à tester sans extension prématurée.

## 5. Architecture des registres K, L, annexe et M

### K. Registre conceptuel dynamique

Fonction :

- classer les concepts par statut, non par dignité ontologique ;
- indiquer usages autorisés, usages interdits, dépendances, critères d'échec et sources.

Catégories minimales :

- concepts de base ;
- relations graduées ;
- dimensions d'analyse ;
- temporalités ;
- devenirs épistémiques ;
- concepts régionaux ;
- concepts suspendus ;
- concepts retirés ;
- critères de gain et d'échec.

Interdits :

- carte canonique close ;
- ancienne taxonomie v12 ;
- hiérarchie physique vers réflexif ;
- synonymie par voisinage lexical.

### L. Voisinages

Fonction :

- reconstruire les traditions et auteurs dans leurs propres problèmes ;
- déterminer ce que le cadre emprunte, reformule, contredit ou rend redondant.

Relations à distinguer :

- emprunt ;
- convergence ;
- concurrence ;
- correction ;
- objection ;
- redondance.

Interdits :

- présenter les voisins comme des versions incomplètes du cadre ;
- attribuer à un auteur une formule forgée par le cadre ;
- défendre l'originalité avant comparaison.

### Annexe généalogique

Fonction :

- élargir la cartographie des filiations et objections ;
- séparer affinité, source, analogie, correction et conflit.

Interdits :

- collection d'affinités vagues ;
- généalogie servant uniquement à légitimer le cadre.

### M. Migrations

Fonction :

- reconstruire les migrations de vocabulaire dans le corpus ;
- empêcher les remplacements globaux ;
- documenter ce qui change lorsqu'un mot change.

Catégories minimales :

- renommage ;
- restriction ;
- élargissement ;
- fragmentation ;
- fusion ;
- régionalisation ;
- requalification modale ;
- suspension ;
- abandon ;
- conservation historique.

Interdits :

- migration fractale ;
- table de synonymie automatique ;
- destination canonique présupposée.

## 6. Dépendances transversales

| Dépendance | Effet architectural |
|---|---|
| A avant B | le vocabulaire doit être présenté comme outil d'enquête, non comme ontologie déjà acquise |
| B avant C | la matrice dépend de distinctions terminologiques minimales |
| C avant D à G | les relations, temporalités, échelles et devenirs épistémiques sont des dimensions croisables, non une succession |
| D avant E | les temporalités des structures ne doivent pas présupposer une relation constitutive |
| E avant F | la persistance doit être distinguée avant d'analyser ses profils de résolution |
| F avant G | il faut distinguer accès, visibilité et mécanisme avant de diagnostiquer recouvrement ou réification |
| G avant H | le test méthodologique doit savoir quels devenirs épistémiques il peut diagnostiquer |
| H avant I et J | héritages et concepts régionaux doivent être évalués par gain, seuils et cas négatifs |
| K après phase 2 mais avant réécriture | K sert de contrôle provisoire des statuts conceptuels |
| L et annexe avant K final | les voisinages peuvent modifier le statut ou la non-redondance des concepts |
| M après K final | la migration dépend des statuts conceptuels stabilisés |

## 7. Concepts autorisés transversalement sous condition

Ces concepts peuvent circuler entre sections seulement si leur statut est explicitement indiqué :

- coupe : terme de travail, jamais invariant acquis ;
- constitution : effet fort à démontrer ;
- co-constitution : hypothèse maximale ;
- transversalité : hypothèse testée, non propriété ;
- sédimentation : régionale ;
- recouvrement cognitif : restreint ;
- réflexivité critique : limitée aux pratiques capables d'explicitation et révision ;
- analogie : ouverture d'enquête, jamais preuve ;
- gain : évalué relativement à une reconstruction indépendante ;
- échec : possibilité réelle.

## 8. Concepts interdits comme prémisses générales

Les éléments suivants ne doivent pas structurer la v13 :

- toute coupe est constitutive ;
- co-constitution universelle ;
- cascade physique-biologique-pré-thématique-méthodologique-réflexive ;
- gradient des rapports à soi ;
- destins universels ;
- porosité comme phase commune ;
- solidification comme catégorie transversale ;
- sédimentation universelle ;
- variation locale comme signature du réel ;
- légifération universelle ;
- migration fractale ;
- quantum comme preuve du noyau ;
- remplacement lexical global.

- de texte définitif pour A à J ;
- de K provisoire ;
- de reconstruction de L ;
- d'annexe généalogique ;
- de registre M ;
- de synchronisation du manuscrit.
