## Travail effectué

La phase 5 a créé deux fichiers :

1. `L_voisinages.md`, consacré aux confrontations décisives ;
2. `Annexe_genealogique.md`, consacrée à la cartographie élargie des sources, voisins, concurrents et correctifs.

## Fonction stabilisée de L

L devient un test de non-redondance philosophique.

Il ne défend pas l'originalité du cadre. Il demande ce qui reste du cadre après confrontation avec ses voisins.

Relations distinguées :

- source généalogique ;
- emprunt ;
- convergence ;
- reformulation ;
- restriction ;
- décomposition ;
- concurrence ;
- correction reçue ;
- incompatibilité ;
- redondance.

Voisins traités comme confrontations décisives provisoires :

- Barad ;
- Simondon ;
- Malabou.

Voisins à instruire prioritairement :

- Husserl ;
- Maturana et Varela ;
- Canguilhem ;
- Foucault ;
- Cassirer ;
- réalisme structural ;
- Stengers ;
- Hacking.

## Fonction stabilisée de l'annexe

L'annexe devient une cartographie élargie.

Elle distingue :

- généalogie effective ;
- voisinage conceptuel ;
- épreuve comparative ;
- concurrence théorique ;
- correctif critique ;
- analogie formelle ;
- incompatibilité.

Elle organise les entrées par constellations problématiques plutôt que par grands blocs homogènes.

## Contrôles de régression

La phase 5 ne doit pas :

- présenter les voisins comme des versions incomplètes du cadre ;
- utiliser la généalogie comme preuve de validité ;
- défendre l'originalité avant comparaison ;
- homogénéiser des courants hétérogènes ;
- attribuer à un auteur une formule du cadre ;
- traiter une ressemblance comme une filiation ;
- maintenir dans l'annexe publique un réservoir interne de concepts suspendus ;
- finaliser K avant L et l'annexe.

## Choix de rédaction

Les deux fichiers ont été rédigés en positif comme état courant.

Les anciennes formulations invalidées par les audits sont conservées comme contrôles dans le présent registre plutôt que reprises comme argumentation négative dans les fichiers principaux.

## Risques restants

- Les fiches d'auteurs restent provisoires et devront être renforcées par sources primaires avant usage public.
- La hiérarchie finale des voisins structurants n'est pas encore fixée.
- Cassirer, réalisme structural, Hacking et Stengers nécessitent des dossiers plus complets.
- K ne peut pas être finalisé sans intégrer les effets de L et de l'annexe.
- M devra documenter les emprunts, restrictions et abandons révélés par cette phase.

## Suite autorisée

Après validation explicite de la phase 5, la suite autorisée est la phase 6 : finalisation de K.
